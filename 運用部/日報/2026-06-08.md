### 今日の学び

- LPは機能よりも「最初の3秒で何者か伝わるか」が重要。
- ウイスキーのような嗜好品は、スペックよりも余韻・時間・飲む場面の設計が価値になる。
- 画像、コピー、CTA、法務配慮を同時に見るには、LP専用のGenius Councilが必要。
- 高級LPは、情報量を増やすよりも、写真・中央軸・余白・改行ルールを揃える方が効く。
- 見出しに句読点を入れない。本文は意味の切れ目で改行する。単語途中の改行は避ける。
- Hero, Land, Craft, Taste, Philosophy, CTA の順で、視覚と文脈を少しずつ深めるとブランドの密度が上がる。
- CTA は横に広げず、中央列で縦に積むと重心が安定する。
- モバイルはPCの縮小版ではなく、別のタイポと改行ルールを持たせたほうが崩れにくい。
- セクションごとに本文サイズと行間を揃えると、見切れが減り、ページ全体の統一感が出る。

### LP作成ルール

| ルール | 意図 |
| --- | --- |
| 商品写真を主役にする | ブランドの温度、価格帯、質感を最初に伝えるため |
| ページ全体を中央軸に揃える | 左右に情報が割れた SaaS 感を避けるため |
| 見出しに句読点を入れない | 高級感と一体感を保つため |
| 本文は句読点の後で改行する | 単語途中・文途中の不自然な折返しを避けるため |
| 同じ画像を使い回さない | セクションごとに役割を持たせるため |
| CTA は縦積みで中央に置く | フォームの重心を崩さないため |
| 年齢確認と注意文を入れる | 酒類LPとして最低限の配慮を満たすため |
