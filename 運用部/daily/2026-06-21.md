## 実装内容
- 都市71(地域別グループ選択)。相手の現在ローカル時刻＋状態バッジ(🟢ちょうどいい/🟡ぎりぎりOK/🔴寝てるかも)。
- 24時間カラーバンド:横軸=自分の時間(左0時→右24時)、色=相手の状態、6/12/18時の目盛と「今」マーカー。次に連絡しやすくなる時刻を文章で案内。
- 相手の生活リズム(起床/就寝)設定で色判定を調整。カードタップで編集・削除。曜日は日本語表示。
- PWA(manifest + Service Worker、network-first)。データは端末内localStorageのみ・外部送信なし。

## 技術メモ
- スタック: Vanilla JS(単一`index.html`・ビルドなし)/ `Intl.DateTimeFormat`(DST・30/45分ズレ自動対応)/ localStorage / PWA / Vercel。正本 `~/dev/when-to-ping/`(100day repo外)
- タイムゾーン: 各都市の現在時刻を `formatToParts` で取得し小数化。時差は (相手frac − 自分frac) から算出。深夜またぎの就寝判定・半時間ズレを Node で検証
- 課金: AI API 不使用＝Anthropic クレジット **ゼロ**(claude -p / Agent SDK / クラウド未使用)。Vercel 無料枠
