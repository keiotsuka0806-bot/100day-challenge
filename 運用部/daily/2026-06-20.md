## 実装内容（シーン別UX）
作成画面の先頭で「何を決めたい？」をカード選択 → 専用フォームに切替の4シーン:
- **❓ 質問を作る**: 単一選択（複数不可）・%表示。クイック入力（はい/いいえ・◯/✕・A/B/C/D）で選択肢を即入力
- **✅ 複数選択アンケート**: 複数選択・人数表示。**参加者が自分の選択肢を追加**できる（最初の一杯を各自入力→全体集計）
- **📅 日程を決める**: 行ける日を複数選択・最多に◎。候補日は**範囲一括追加＋平日/土日フィルタ**
- **⭐ 星評価**: ★1〜5・平均点とばらつき
- 横断: **🙈匿名／🙋記名**切替（記名は投票者名を表示）、主催者の「🗳 自分も投票する」＋戻る導線

## 技術メモ
- スタック: Vanilla JS（ビルドなし）/ Firebase Firestore / Vercel / qrcode-generator / PWA。正本 `開発部/qr-live-poll/`
- データ: `poll = { type, question, named, allowAdd, options:[{text,votes,names?}], voters }`。複数選択は `vote(id, indices, name)` で一括加算

- **デプロイの罠**: `firebase-config.js` はGit除外 → そのままだとVercel本番に届かずデモのまま。`.vercelignore` で“除外しない”ことで本番に届けた
- 課金: AI API 不使用＝Anthropic クレジットゼロ。Firebase/Vercel 無料枠
