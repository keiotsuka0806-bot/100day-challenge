# タイトル案（3つ）

1. カメラもAIも使わず、「今日なにしてたっけ？」を消す道具を作った
2. 自分の1日を、勝手にタイムラインにする——非エンジニアの“安い道具”づくり
3. 「自分用」と決めた瞬間、設計がきれいになった話

---

## リード（無料部分の冒頭）

夜になると、よくこうなる。「今日って結局、なにしてたっけ？」
やったはずなのに、時間ごとに何をしていたかを正確に思い出すのは、意外と難しい。

そこで、自分の1日のPC作業を勝手に記録して、時間ごとのタイムラインにしてくれる道具を作ることにした。派手な機能はない。でも、毎日効く。今日はその「作り方の判断」の話をしたい。何を入れて、何を入れなかったか。そこに、いちばん考えたことが詰まっている。

---

## 本文

### 最初はカメラを考えた。でもやめた

はじめは、カメラで自分を撮って「集中しているか」を測る案を考えていた。いかにも未来っぽい。でも調べていくと、本当に効くのはカメラじゃないと分かってきた。

人が1日に何をしていたかを語るのは、顔より「文脈」だ。どのアプリを開いていたか、何のファイルを触っていたか、どんなページを調べていたか——この情報の方が、「自分が何をしていたか」をよっぽど正確に表す。たとえば同じ作業を45分続けていれば集中、アプリを次々切り替えていれば調べ物や迷い、というふうに、振る舞いから中身が見えてくる。

だからカメラはやめた。代わりに、1分ごとに「今いちばん手前にあるアプリと、その画面のタイトル」をメモするだけの、地味な道具にした。監視カメラというより、勝手につけてくれる行動メモ帳に近い。

### お金をかけない作り方

ここで大事にしたのが「安く、壊れにくく」だった。

いちばんお金がかかるのは、外部のAIサービスに有料でお願いする窓口（API）を使うところだ。だからそこは、最初は一切使わないことにした。記録するのは、ただの文字。1分に1行のメモなら、容量は1日およそ150キロバイト。1年ためても、スマホの写真10数枚ぶんにしかならない。パソコンの容量を圧迫して使えなくなる心配は、まず起きない。

AIに頼むのは、夜に1回だけ。できあがった記録を、対話の画面に貼って「今日を要約して」と言う。これなら、毎日使っても費用はほとんどかからない。派手なAI連携を最初から組み込まないこと——ここが、安く作るいちばんのコツだった。

### 「自分用」と決めた瞬間、設計がきれいになった

作っているうちに欲が出た。「これ、他の人にも使ってもらえないかな」。
でも、他人に使ってもらうのは、段階を上げるほど責任が重くなる。最終的に、みんなの活動記録をネット上のサーバーに集める形まで行くと、私は他人の超センシティブな情報を預かることになる。今の私に、その責任はまだ持てない。

だから「まずは自分用」を選んだ。そして設計の芯を、「記録は本人のパソコンから一歩も外に出さない」に置いた。これはケチった結果じゃない。いちばん安全で、いちばん信頼できる形でもある。データが外に出ないなら、そもそも漏らしようがない。「自分用」と腹をくくった瞬間、迷っていた設計が、すっと一本に決まった。

作る前に「これは他人に配るものか、自分の繰り返し作業か」を分ける。当たり前のようで、つい「作れるから作る」になりがちなところを、今日はちゃんと立ち止まれた。

### 毎朝の起動を、忘れないように自動にした

道具ができても、「毎朝それを手で起動するの、絶対に忘れる」と思った。

そこで、Macに最初から備わっている“予約実行”の仕組みを使い、パソコンを立ち上げたら自動で記録が始まるようにした。新しい有料サービスは何も足していないので、ここでも費用はかからない。朝は、もう何も打たなくていい。

夜は、私がいつも書いている日報の流れに「記録の書き出し」をくっつけた。「日報を書いて」と頼めば、今日のタイムラインが勝手に材料として流れ込む。朝は自動、夜はいつも通り。それだけで1日が記録になる仕組みになった。

> （参考までに、手で動かすときはこの2行だけ）
> 朝　npm start で記録開始
> 夜　npm run export で今日のタイムラインを書き出し

### 派手じゃないけど、毎日使う本命

カメラを足した方が見た目は派手だったと思う。でも、毎日使うのはきっとこっちだ。

「今日なにしてたっけ？」が消えるだけで、1日の手応えが変わる。何に時間を使い、どこで迷っていたかが、あとから自分の目で見える。派手さより、毎日効くこと。今日いちばん納得したのは、その地味な価値の方だった。

気に入ったら、次はこの道具を、同じように作っている仲間にも渡せる形（コードの公開）にしていきたい。でも順番は、まず自分でしばらく使ってみてから。本当に毎日使いたくなる道具かどうかは、自分が試さないと分からない。
