## 1. 習慣トラッカー「StreakBoard」
**概要**: 毎日の習慣をチェックするだけのシンプルなアプリ。連続達成日数（ストリーク）をビジュアルで表示。  
**技術**: Vanilla JS + localStorage（Firebaseなし）  
**UI**: グリッド形式のカレンダー × GitHub草アイコン  
**MVP機能**:
- 習慣を3〜5つ登録
- 毎日チェックするだけ
- ストリーク数をカウント表示
- PWA対応（ホーム画面から起動）

## 2. ランダム晩ごはん決定機「今日なに食べる？」
**概要**: 「また今日も何食べるか迷う」問題を解決。条件（和食/洋食/時間/気分）を選ぶと1品がドーンと提案される。  
**技術**: Vanilla JS + 料理データJSON  
**UI**: スロットマシン風アニメーション  
**MVP機能**:
- カテゴリ選択（和食・洋食・中華・なんでも）
- 調理時間フィルター（15分以内 / 30分以内 / なんでも）
- ランダム表示 + 再抽選ボタン
- 料理名・材料・簡単な作り方を表示

## 3. コード読書メモ「CodeLog」
**概要**: OSSや業務コードを読むとき、気づきや疑問をその場でメモるアプリ。ファイル名・行番号つきで残せる。  
**技術**: Vanilla JS + Firebase Firestore  
**UI**: シンプルなメモ帳スタイル  
**MVP機能**:
- ファイル名 + 行番号 + メモの3点セットで記録
- タグ付け（疑問・気づき・TODO・重要）
- プロジェクト名でグループ化
- Firebase Auth（Googleログイン）

---

## 4. 写真日記「1日1枚」
**概要**: 毎日1枚だけ写真を投稿できる日記アプリ。多投稿禁止・1行コメントのみ。シンプルさが売り。  
**技術**: Vanilla JS + Firebase Auth + Firestore（base64保存）  
**UI**: 写真が大きく映えるフィード  
**MVP機能**:
- 1日1枚の写真 + 1行テキスト
- 過去の投稿をカレンダーで振り返り
- 2枚目を投稿しようとするとブロック
- PWA対応

---

## 5. ポモドーロ「PomoCafe」
**概要**: カフェの環境音（雨音・喫茶店・焚き火）と組み合わせたポモドーロタイマー。音と時間で集中モードへ。  
**技術**: Vanilla JS + Web Audio API  
**UI**: 落ち着いたカフェ風デザイン  
**MVP機能**:
- 25分 / 5分のポモドーロサイクル
- BGM選択（雨音・カフェ・焚き火 — CDN音源）
- 残り時間の大きな表示
- 終了時の通知（Notification API）
