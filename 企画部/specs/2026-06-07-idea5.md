# QuickFlash — テキストをAIがフラッシュカードに変換する暗記ツール

## 概要
教科書や資料のテキストを貼り付けると、AIが自動でQ&Aフラッシュカードを生成する。カード表示→めくり→自己採点のシンプルな暗記フローを提供する。

## MVP機能リスト
1. テキストエリアにテキストを貼り付け、「カード生成」ボタンでClaude APIへ送信
2. AIが5〜10枚のQ&Aカードを生成して返却
3. カードをクリックでめくるアニメーション（CSS 3D flip）
4. 「知ってた」「知らなかった」の自己採点ボタン
5. セッション結果（正解率・総カード数）をLocalStorageで保持

## 技術スタック
- フロントエンド: Vanilla JS + CSS 3D transform
- データ: LocalStorage
- 特殊ライブラリ: Claude API（Vercel Serverless Functionでキー隠蔽）

## 難度
**低（1–2h）**
APIは既存のVercel Functionパターン流用可。カードめくりアニメーションはCSS transformで実現。学習系ツールとして反響が期待できる。
