# TechCuratorJP 仕様書

## 概要
Hacker NewsとProduct Huntのデイリーハイライトを自動収集し、Claude APIが日本語でキュレーションするPWA。「今日のテック界隈で何が起きているか」を3分で把握できる朝のダッシュボード。

## ターゲットユーザー
英語の技術ニュースを追いたいが時間がない日本人エンジニア・個人開発者。毎朝手軽にトレンドを確認したい人。

## MVP機能一覧
- [ ] HN API（無料）からトップ10記事を取得
- [ ] PH API or スクレイピングでデイリートップ5取得
- [ ] Claude APIが「3行要約 + 開発者への示唆」を日本語で生成
- [ ] カード形式で表示（HNセクション / PHセクション）
- [ ] PWA対応（ホーム画面追加可能）
- [ ] ローカルキャッシュ（localStorage）で再読込不要

## 将来拡張（MVP後）
- プッシュ通知（毎朝8時自動更新）
- カテゴリフィルター（AI / Security / Frontend / etc.）
- お気に入り記事の保存

## 技術スタック
- フロントエンド: Vanilla JS
- API: HN Firebase API（無料） + Claude API（Serverless Function）
- バックエンド: Vercel Serverless Function
- ホスティング: Vercel
- 特殊ライブラリ: なし

## データモデル
- localStorage: `{ date: "2026-06-08", hn: [...], ph: [...], summary: "..." }`
- 同日キャッシュがあれば API呼び出しをスキップ

## 画面構成
- ヘッダー: 日付 + 「今日のテック」タイトル + 更新ボタン
- セクション1: HN ハイライト（カード5枚）
- セクション2: PH 注目プロダクト（カード3枚）
- フッター: 最終更新時刻
