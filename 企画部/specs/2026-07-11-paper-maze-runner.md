# 紙迷路ランナー（PaperMazeRunner）

## 概要
紙にペンで描いた迷路をカメラで撮ると、その迷路が画面の中で「遊べるゲーム」になり、スマホを傾けてボールを転がして遊べる。描く→撮る→動き出す、の10秒。

## MVP機能
- カメラ撮影→二値化で線を壁として検出（端末内処理・AI不使用）
- DeviceOrientation（傾き）で玉転がし物理（Canvas自前物理）
- スタート/ゴール自動判定（丸=スタート、星/二重丸=ゴール等の簡易ルール）＋クリアタイム共有カード

## 技術スタック
- Vanilla JS / Canvas / DeviceOrientation API / PWA
- AI・サーバー不要（完全無料・オフライン動作）

## 難度
**中〜高（半日〜1日）**

## 独自性
「物理お絵かきゲーム（Physics Draw系）に似ているが、画面に描くのではなく**現実の紙とペンで描いたものが動き出す**＝子どもの落書きがそのままゲームになる点が違う」

## 収益の芽
なし（笑える/すごい枠。シェアで届く方が価値）
