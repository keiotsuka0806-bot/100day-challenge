## コンセプト

**「遊びながらソムリエの知識が自然に身につくボードゲーム」**

ゲームを通じて以下が学べる：
- ワイン産地と産出品種の対応（地理）
- 品種ごとの味わい・香りの特徴
- ワインと料理のペアリング
- 醸造方法・格付け制度
- 実際のソムリエが行う判断

---

## ゲームの骨格（3つのゲームの融合）

| 要素 | 元ゲーム | VinRouteでの役割 |
|------|---------|----------------|
| すごろく移動 + マスイベント | 桃鉄 | 移動のたびに学びが発生する仕組み |
| リソース管理 + パッシブ収入 | カタン | 畑→醸造所の建設による戦略性 |
| 目的地レース + 妨害キャラ | 桃鉄 | 競争の緊張感・逆転要素 |
| 知識クイズ | VinRoute独自 | 学習の核心。知識がリソースに直結 |

## マップ設計

### 産地（15ヶ所）
産地はヘックスタイルで表示。各産地間に2〜3の「ルートマス」を配置。

```
産地 ─[📖]─[🍷]─ 産地
  │                  │
[🎲]              [🛒]
  │                  │
産地 ──[📖]── 産地
```

### マスの種類（5種類）

| マス | アイコン | 内容 |
|------|---------|------|
| **産地マス** | 🏰 | 産地カード表示 → クイズ → リソース獲得 |
| **豆知識マス** | 📖 | 通過で1行フラッシュ、止まると詳細表示 |
| **試飲マス** | 🍷 | ブラインドテイスティング問題。正解で試飲カード入手 |
| **市場マス** | 🛒 | リソースの売買・プレイヤー間交換 |
| **イベントマス** | 🎲 | 豊作・霜害・評論家来訪などランダムイベント |

## 学習システム

### 設計原則
- 知識ゼロでも遊べること
- 知識があると有利になること
- 正解・不正解どちらでも学びが起きること

### 産地カード（着地時に必ず表示）
```
┌─────────────────────────┐
│ 🏰 ブルゴーニュ / フランス   │
│                           │
│ 主要品種：ピノ・ノワール      │
│ スタイル：単一品種のみが伝統   │
│ 特徴：世界最高のピノ産地      │
└─────────────────────────┘
```
カードには答えは書かない（品種名は書くが、その品種の「特徴」は書かない）

### クイズの種類（7種類）

| # | 種類 | 例 | 難易度 |
|---|------|---|--------|
| 1 | **品種の性質** | 「ピノ・ノワールの特徴は？」 | 中 |
| 2 | **ペアリング** | 「このワインに合う料理は？」 | 易 |
| 3 | **ブラインドテイスティング** | 「この説明の品種は？（香り・色から）」 | 難 |
| 4 | **お客様シナリオ** | 「渋みが苦手なお客様には？」 | 易 |
| 5 | **産地当て（逆引き）** | 「急斜面・石板土壌・甘口白。この産地は？」 | 難 |
| 6 | **なぜ？** | 「ボルドーがブレンドする理由は？」 | 中 |
| 7 | **ラベル読解** | 「"Grand Cru"の意味は？」 | 中 |

### リソース獲得ルール
- クイズ正解：そのマスに対応したリソース × 2
- クイズ不正解：リソース × 1（ペナルティなし）
- 正解・不正解後：必ず解説を1行表示（これが学習の核心）

## ターン構造（確定）

```
① サイコロ1個（1〜6）

② マスを1つずつ移動
   ・通過マス：豆知識が1行フラッシュ表示
   ・止まったマス：そのマスのイベント発動
   ・産地マス着地：産地カード → クイズ → リソース獲得

③ アクション（任意・1つ）
   ・醸造する（リソース消費 → VP獲得 + 豆知識表示）
   ・畑を建てる（以降パッシブ収入）
   ・カードを使用

④ 他プレイヤーのターン中も、自分の畑からパッシブ収穫

⑤ 4ターンごと「収穫祭」
   ・全畑が一斉収穫
   ・新しい目的地カード2枚を公開
   ・評論家が最下位プレイヤーに移動
```

## 競争・逆転要素

### 目的地レース（桃鉄の目的地）
- 毎シーズン開幕に「大賞産地」「小賞産地」を公開
- 大賞産地に最初に到達 → その産地の独占醸造権（他プレイヤーはそこで醸造不可）
- 小賞産地に最初に到達 → リソース × 3ボーナス

### 辛口評論家（桃鉄のボンビー）
- 常に最下位プレイヤーに憑依
- 憑依中：醸造するたびVP -1
- 振り払う方法：クイズに2連続正解

### 特殊カード（桃鉄の特急）
- 試飲カード：クイズをスキップしてリソース最大値獲得
- 特急チケット：任意の産地に一瞬で移動
- 評論家撃退カード：評論家を別プレイヤーに転送

## 醸造（VP獲得）

醸造時に「ワイン豆知識カード」を表示 → 知識の定着を促す

| ワイン | 必要リソース | VP | 表示される豆知識 |
|--------|------------|-----|----------------|
| ボルドー・ブレンド | カベルネ×2 + ブレンド×1 | 3 | 「ボルドーはメドックとポムロールで主要品種が異なる」 |
| ブルゴーニュ | ピノ×3 | 4 | 「単一品種100%が伝統。産地内の格付けが細かく分かれる」 |
| シャンパーニュ | シャルドネ×2 + リースリング×1 | 3 | 「瓶内二次発酵（メトード・トラディショネル）で泡を作る」 |
| ロゼ | ピノ×1 + シャルドネ×1 | 2 | 「ロゼは赤ブドウの短時間浸漬で色を付ける」 |
| グランクリュ | 全5種×1 | 6 | 「グランクリュ（特級畑）はブルゴーニュで33ヶ所のみ認定」 |
| 甲州 | リースリング×1 + ブレンド×2 | 2 | 「甲州は日本固有品種。和食との相性が高い」 |

## 現行コードとの差分

現在 `開発部/wine-board-game/` に実装済みだが、以下が大きく変わる：

| 要素 | 現行 | v2 |
|------|------|-----|
| 移動 | 産地間をステップ移動 | 産地＋ルートマスをすごろく移動 |
| クイズ | ランダム発動・任意 | 産地着地で必ず発動、種類7種 |
| 産地カード | なし | 着地時に必ず表示 |
| 目的地 | 弱い | シーズンごとにレース化 |
| 評論家 | 簡易実装 | 最下位憑依・振り払いロジック |
| マップ | ヘックスのみ | ヘックス＋ルートマス |
