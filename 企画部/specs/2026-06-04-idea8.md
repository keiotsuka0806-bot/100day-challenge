# 週報ジェネレーター 仕様書

## 概要
今週やったこと・学んだこと・来週やることをフォームに入力するだけで、そのまま使える週報テキストを自動生成するアプリ。コピペでSlackやNotionに貼れる。（トレンド反映: 開発の「記録・発信」ニーズ）

## ターゲットユーザー
週報・進捗報告を書く個人開発者・フリーランス・チーム開発者

## MVP機能一覧
- [ ] 今週やったこと（箇条書き入力）
- [ ] 学んだこと（箇条書き入力）
- [ ] 来週やること（箇条書き入力）
- [ ] テンプレート選択（Slack形式・Notion形式・シンプル形式）
- [ ] ワンクリックでテキストをコピー
- [ ] localStorage で下書き自動保存

## 将来拡張（MVP後）
- 過去の週報履歴
- Firebase同期でPCとスマホ共有
- X（Twitter）への直接投稿

## 技術スタック
- フロントエンド: React + Vite
- データ: localStorage
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## 画面構成
- 1画面: 入力フォーム（3セクション）+ テンプレート選択 + 生成結果プレビュー + コピーボタン
