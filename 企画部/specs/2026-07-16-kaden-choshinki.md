# 家電の聴診器（Kaden Choshinki）

## 概要
「洗濯機の音、先週より変じゃない？」を記憶でなく記録で判断する。家電の動作音をブラウザで録音し、波形・周波数の特徴を端末内で保存。時系列で比較して「音が変わった」を可視化し、修理相談時に聞かせられる音のカルテを作る。

## MVP機能
- 家電ごとの録音カルテ（Web Audio APIで波形＋スペクトログラム表示・端末内保存）
- 前回録音との差分表示（音量・主要周波数の変化）
- 症状チェックリスト（異音の種類×家電の人手キュレーション）＋修理窓口メモ

## 技術スタック
- Vanilla JS / Web Audio API / Canvas / localStorage / PWA（AI不使用・コストゼロ）

## 備考
研究起点: 世界モデル（音による状態推定）の家庭翻訳。ただし「音が変わった」から先の診断はできず、行動につながる出力が弱い（記録止まり）リスク。
