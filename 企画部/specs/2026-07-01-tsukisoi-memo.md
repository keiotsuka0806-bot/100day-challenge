# 通院の付き添いメモ（Tsukisoi）

## 概要
病院の診察を音声で録っておく（or その場でメモ）と、AIが「先生が言った要点／次回まで／薬／受診の理由」を構造化し、離れて暮らす家族にそのまま共有できる短いカードにする。

## MVP機能
- 音声/テキスト入力→診察内容を4項目（要点/次回/薬/理由）に構造化
- 家族に送る用の短い共有カード（LINEに貼れる）
- 断定を避け「先生に確認」を促す注記

## 技術スタック
- Vanilla JS / PWA / Web Speech API / OpenAI API(gpt-4o-mini)

## 難度
**中（半日）**

## 独自性
「kensa-yomi/doc-decodeが書類を読むのに対し、診察室の会話を家族に橋渡しする」。声入力・他者共有が主目的で形が違う。

## 20年後の視点
医療記録が自動で家族に届く時代でも「その場の不安を誰かに渡せる」価値は残る。

## 備考
研究起点: 長文コンテキストの構造化要約＋医療ドメインの慎重さ。
