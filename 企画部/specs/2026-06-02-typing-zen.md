# TypingZen 仕様書

## 概要
日本語の名言・ことわざをタイピングする練習アプリ。落ち着いたデザインで「ゆっくり正確に」を重視するタイピング体験を提供する。

## ターゲットユーザー
タイピング速度より正確性を上げたい人。日本語入力に慣れたい初中級者。

## MVP機能一覧
- [ ] 名言・ことわざをランダム表示
- [ ] 入力中にリアルタイムで正誤を色分け表示
- [ ] 正解率・所要時間を結果表示
- [ ] ジャンル選択（名言・ことわざ・プログラミング格言）
- [ ] PWA対応

## 将来拡張（MVP後）
- スコアランキング（Firebase）
- 自分でテキストを登録
- タイムアタックモード

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: 名言JSONファイル（100件程度ハードコード）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```json
{
  "quotes": [
    {
      "text": "千里の道も一歩から",
      "category": "ことわざ",
      "reading": "せんりのみちもいっぽから"
    }
  ]
}
```

## 画面構成
1. **ホーム** — ジャンル選択 + スタートボタン
2. **タイピング画面** — テキスト表示 + 入力フォーム + 進捗バー
3. **結果画面** — 正解率・WPM・所要時間 + もう一度ボタン
