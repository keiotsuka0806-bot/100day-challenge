# 割り勘計算機「WarikanPro」仕様書

## 概要
複数人での飲み会・旅行の費用を一括管理・精算できるアプリ。「誰が何を払ったか」を入力すると最小の送金回数で精算方法を計算する。

## ターゲットユーザー
グループでの飲み会・旅行・共同購入で費用精算が面倒な人。

## MVP機能一覧
- [ ] メンバー追加（名前のみ）
- [ ] 支払い記録（誰が・いくら・何のため）
- [ ] 最小送金回数での精算計算
- [ ] 精算結果を「AさんはBさんに1,200円送る」形式で表示
- [ ] URLシェア機能（localStorageのデータをBase64エンコードしてURLパラメータに）

## 将来拡張（MVP後）
- Firebase でセッション共有（リアルタイム同期）
- 領収書写真添付
- LINE通知連携

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: localStorage + URLパラメータ（Firebaseなし）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```json
{
  "session": {
    "name": "2026年忘年会",
    "members": ["田中", "鈴木", "山田"],
    "payments": [
      { "payer": "田中", "amount": 6000, "desc": "居酒屋代" },
      { "payer": "鈴木", "amount": 2000, "desc": "2次会" }
    ]
  }
}
```

## 画面構成
1. **ホーム** — セッション名 + メンバー入力
2. **支払い入力** — 払った人・金額・説明の追加
3. **精算結果** — 送金先一覧 + シェアボタン
