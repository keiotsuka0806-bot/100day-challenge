# 地震体感シミュレーター（QuakeSense）

## 概要
震度0〜7のゆれをCSSアニメーション＋Web Vibration APIで体感できるインタラクティブ啓発ツール。日本に暮らす外国人や子どもに地震スケールを直感的に伝える。

## MVP機能
- 震度スライダー（0〜7）を動かすとページ全体が対応する揺れ幅でアニメーション
- 各震度の体感説明文（日本語・英語）を同時表示
- スマホ実機では Vibration API で振動フィードバック

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし（静的コンテンツ）
- 特殊ライブラリ: なし（CSS animation + Web Vibration API）

## 独自性
気象庁震度解説ページに似ているが、「数字でなく身体で理解する」インタラクティブ体感版として差別化。外国人観光客・留学生への防災教育に特化。

## 20年後の視点
気候変動に伴う地殻活動リスクの増加が予測される2045年、防災リテラシー啓発ツールの社会的価値は格段に上がる。

## 備考
気象庁の震度階級関連解説表をベースに実装。CSSのtranslate/rotateアニメーションの強度を震度ごとに調整するだけで実現可能。多言語対応（ja/en）で国際的な配布も視野に入れる。
