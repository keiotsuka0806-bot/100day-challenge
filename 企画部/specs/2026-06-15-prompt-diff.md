# プロンプト差分道場（PromptDiff）

## 概要
2つのプロンプトを入れると、出力がどう変わりそうかをClaudeが解説し、改善版を提案する。

## MVP機能
- プロンプトA/Bを入力
- 出力傾向の違いを解説
- 統合した改善版を提案

## 技術スタック
- Vanilla JS / Claude API

## 難度
**中（半日）**

## 独自性
「TitleDojo等のプロンプト系に似ているが、“2案の差分比較”に特化する点が違う」

## 20年後の視点
2045年、プロンプトは人が書かなくなり、差分を見る行為自体が懐古趣味になる。
