# タブ整理 Chrome拡張（TabNinja）

## 概要
開いている大量のタブをドメイン別・最終アクセス時刻別に自動グループ化し、一括クローズ・ブックマーク化できるChrome拡張。

## MVP機能
- ポップアップ画面でタブ一覧をドメイン別グループ表示
- グループ単位で「全て閉じる」「全てブックマーク」ボタン
- 1週間以上アクセスしていないタブをハイライト表示
- タブ数が30以上になったらバッジで警告

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Chrome Storage API
- 特殊ライブラリ: なし
