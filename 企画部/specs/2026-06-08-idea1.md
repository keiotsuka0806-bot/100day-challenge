# SecurityDigest 仕様書

## 概要
Hacker News APIからセキュリティ関連記事を自動収集し、Claude APIが日本語で3行要約するPWA。毎朝開くだけで今日のセキュリティトレンドが把握できる。

## ターゲットユーザー
セキュリティに関心のある開発者・エンジニア。英語記事を読む時間がない人。

## MVP機能一覧
- [ ] HN APIから「security」「vulnerability」「attack」キーワードの上位記事を取得
- [ ] Claude APIで各記事を日本語3行要約（タイトル・要点・開発者への影響）
- [ ] カード形式で一覧表示（元記事リンク付き）
- [ ] 「今すぐ更新」ボタンで再取得
- [ ] PWA対応（manifest + service worker）

## 将来拡張（MVP後）
- キーワードフィルター設定
- 重要度スコアリング
- プッシュ通知（毎朝9時）

## 技術スタック
- フロントエンド: Vanilla JS
- API: HN API（無料・認証不要）+ Claude API（Serverless Function経由）
- バックエンド: Vercel Serverless Function（APIキー隠蔽）
- ホスティング: Vercel
- 特殊ライブラリ: なし

## データモデル
- クライアント側ローカルキャッシュ（localStorage）で最終取得結果を保持
- Serverless Functionがリクエストごとに HN API → Claude API を呼ぶ

## 画面構成
- トップ: カード一覧（タイトル / 日本語要約 / HNリンク）
- ヘッダー: アプリ名 + 最終更新時刻 + 更新ボタン

## デプロイ先
- Firebase project: なし（Vercel）
- URL: https://security-digest.vercel.app（仮）

## 難度・工数
低（2時間）
