# ReceiptLogger — レシート写真をAIが解析して支出を自動記録するアプリ

## 概要
レシートの写真を撮影・アップロードすると、Claude Vision APIが品目・金額・店名・日付を自動抽出して家計簿に記録する。手入力ゼロの支出管理アプリ。

## MVP機能リスト
1. カメラ撮影またはファイルアップロードでレシート画像を入力
2. Claude Vision APIがレシートを解析し、JSON形式で品目・合計・店名・日付を返却
3. 解析結果を確認・修正できる編集フォーム（ワンタップ確定）
4. Firebase Firestoreに記録を保存し、月別支出サマリーを表示
5. カテゴリ別（食費・日用品・交通など）の円グラフ表示（Chart.js）

## 技術スタック
- フロントエンド: Vanilla JS + Canvas API（画像リサイズ）
- データ: Firebase Firestore + Firebase Auth
- 特殊ライブラリ: Claude Vision API（Vercel Serverless）・Chart.js CDN

## 難度
**高（1日フル）**
Vision APIのレシート解析精度に依存するため、プロンプト調整に時間がかかる。Firebase設計（品目単位 vs レシート単位）の意思決定も必要。実用性が高く、継続的に使いたくなるアプリ。
