# AICompanion 仕様書

## 概要
毎日話しかけると会話が蓄積され、AIが自分のことを少しずつ覚えていくシンプルなコンパニオン日記アプリ。「今日どうだった？」という問いから始まる、記録と対話を兼ねた体験。

## ターゲットユーザー
日記を続けたいが続かない人。誰かに話しかける感覚でその日を振り返りたい人。AI Companion市場（2026年$486億・CAGR 31%）の入口として。

## MVP機能一覧
- [ ] Firebase Auth（Google）でログイン
- [ ] チャット形式UI（ユーザー入力 → AI返答）
- [ ] Claude APIが過去の会話コンテキスト（直近10件）を踏まえて返答
- [ ] Firestoreに会話履歴を日付単位で保存
- [ ] 「今日のまとめ」ボタン：Claude APIがその日の会話をひと段落にまとめて表示
- [ ] カレンダービューで過去の対話日を確認

## 将来拡張（MVP後）
- 感情分析グラフ（週次メンタルヘルスサマリー）
- ペルソナ設定（友達口調 / メンター口調 / 哲学者口調）
- 音声入力対応

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Firebase Firestore + Firebase Auth
- バックエンド: Firebase Functions（Claude API呼び出し・APIキー隠蔽）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: Firebase SDK（CDN）

## データモデル
```
users/{uid}/conversations/{date}/
  messages: [
    { role: "user", text: "...", timestamp },
    { role: "assistant", text: "...", timestamp }
  ]
  summary: "..."
```

## 画面構成
- メイン: チャット画面（LINEスタイル）
- サイドバー or 下部: カレンダー（対話済み日をハイライト）
- ヘッダー: 今日の日付 + 「今日のまとめ」ボタン
