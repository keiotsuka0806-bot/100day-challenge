# BreakTimer — シンプルポモドーロタイマー

## 概要
25分集中・5分休憩のポモドーロタイマー。終了時に気持ちいい音が鳴り、今日の集中セッション数を記録する。

## MVP機能リスト
1. 25分 / 5分カウントダウン表示（クリックでスタート）
2. タイマー終了時にWeb Audio APIで短い通知音を生成
3. 今日の完了セッション数を表示（🍅 × N）
4. 画面タップでカウント一時停止・再開
5. LocalStorageで今日のセッション数を保持

## 技術スタック
- フロントエンド: Vanilla JS + CSS アニメーション（円形プログレスバー）
- データ: LocalStorage
- 特殊ライブラリ: Web Audio API（ライブラリ不要・ブラウザ標準）

## 難度
**低（1–2h）**
円形プログレスバーのSVGアニメーションだけ少し手間だが、全体は単純なsetInterval実装。
