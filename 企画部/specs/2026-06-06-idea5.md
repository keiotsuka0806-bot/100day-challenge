# ブラウザMIDIビジュアライザー（MIDIViz）

## 概要
MIDIファイルをドラッグ＆ドロップするだけで、ピアノロール風の「落ちてくるノート」アニメーションをブラウザ内でリアルタイム再生するビジュアライザー。

## MVP機能
- MIDIファイル読み込み（D&D + ファイル選択）
- ピアノロール形式のノート落下アニメーション（Canvas描画）
- 再生速度調整・一時停止・シーク
- トラック別カラーリング（最大16トラック）

## 技術スタック
- フロントエンド: Vanilla JS + Canvas API
- データ: なし（ファイルはメモリのみ）
- 特殊ライブラリ: midi-parser-js（CDN）、Tone.js（音声再生）

## 難度
**低（1〜2h）**

## 独自性
MuseScoreやOnlineSequencerに似ているが、インストール不要・アカウント不要でローカルファイルを即座に美しくビジュアライズするシンプルさが違う。

## 20年後の視点
2045年に音楽の視覚体験は空間コンピューティングで立体化されるが、その視覚的文法（音＝形・色・動き）のプロトタイプを作っている。

## 備考
クリエイター向けのSNS映えビジュアル生成にも使える。スクリーンショット保存機能を追加するだけで「MIDIアート」として共有可能になる。
