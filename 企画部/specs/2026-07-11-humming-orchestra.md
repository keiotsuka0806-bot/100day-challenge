# ハミング交響楽団（HummingOrchestra）

## 概要
鼻歌をマイクで拾うと、その場でハモリとストリングスが重なって「自分の鼻歌がオーケストラになる」体験。録音してそのまま人に送れる。

## MVP機能
- マイク入力→ピッチ検出（自己相関）→リアルタイムでコード推定
- Web Audio合成でハモリ＋伴奏を重ねて再生
- 録音→音声ファイル書き出し・共有

## 技術スタック
- Vanilla JS / Web Audio API / PWA（AI・サーバー不要）

## 難度
**高（1日フル）**

## 独自性
「カラオケ採点アプリに似ているが、採点でなく**自分の声が主旋律の合奏が即座に生まれる**点が違う」

## 収益の芽
なし（すごい枠）

## 20年後の視点
2045年、演奏技術がなくても音楽表現ができるのは当たり前で、これはその入口の一つ。

## 備考
研究起点: なし（技術デモ枠と正直に明記）。Keiの音楽鉱脈に自然に重なる1案。懸念: ピッチ検出の品質が体験の成立条件で、1日で「すごい」水準に届かないリスクが大きい。
