# 画像フォーマット変換＆圧縮比較ツール（FormatShift）

## 概要
ブラウザ上でJPEG/PNGをWebP・AVIFに変換し、変換前後のファイルサイズと画質をビジュアルに比較するデベロッパーツール。Canvas APIのみで完結しサーバー不要。

## MVP機能
- 画像ドラッグ&ドロップ → WebP / AVIF に変換
- 変換前後のファイルサイズ・圧縮率を並列表示
- 変換後ファイルのダウンロードボタン

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし（Canvas API + Blob API完結）
- 特殊ライブラリ: なし

## 難度
**低（1〜2h）**

## 独自性
Squooshに似ているが、「変換比較のUIを最小限に絞ったワンクリック版」として素早くWebP化したい開発者向けに特化。

## 20年後の視点
AI生成画像の爆増で2045年のストレージコストは逆説的に上昇する——フォーマット最適化の重要性は今より増す。

## 備考
AVIFはChrome/Firefox対応済み。Canvas.toBlob()でWebP出力、AVIFはlibavif WASMを使うか、対応ブラウザのみ対象にする方針で難度を低に抑える。
