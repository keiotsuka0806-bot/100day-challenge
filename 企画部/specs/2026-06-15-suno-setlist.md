# セットリスト構成（SunoSetlist）

## 概要
作ったSuno曲のタイトルと雰囲気を並べて入れると、Claudeがアルバムの曲順・コンセプト・曲間のつなぎコメントを提案する。

## MVP機能
- 曲名＋雰囲気タグを複数入力
- Claudeが曲順とアルバムコンセプトを提案
- 各曲のつなぎ（流れの理由）を1行ずつ生成

## 技術スタック
- Vanilla JS / localStorage / Claude API

## 難度
**中（半日）**

## 独自性
「SunoRx（プロンプト処方箋）に似ているが、1曲ではなく“曲が溜まった後のアルバム構成”という別レイヤーを扱う点が違う」

## 20年後の視点
2045年、作曲は無限に増え、価値は「並べ方＝物語」に移る。曲順が作品になる。
