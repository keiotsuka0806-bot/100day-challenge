# AIコードレビューアシスタント（CodeWhisper）

## 概要
コードをテキストエリアに貼り付けると、Claude APIが「バグの可能性」「改善提案」「良い点」の3カテゴリでレビューコメントを返すWebツール。

## MVP機能
- コード貼り付けエリア（言語選択: JS / Python / Go / その他）
- Claude APIによる3カテゴリレビュー生成（バグ / 改善 / 良い点）
- レビュー結果のMarkdown表示
- 「もっと詳しく」ボタンで特定コメントを深掘り

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし（ステートレス）
- 特殊ライブラリ: Claude API（claude-sonnet-4-6）、marked.js（Markdown描画）

## 難度
**高（1日フル）**

## 備考
GitHub trending急上昇の「AIスキルフレームワーク」群やProduct Hunt「CodeRabbit」のトレンドを反映。プロンプトエンジニアリング（構造化出力 + JSON mode）の設計がメインの難所。APIキーはサーバーサイド（Cloudflare Workers）経由を推奨。
