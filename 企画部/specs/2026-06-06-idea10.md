# SNS映え食事採点カメラ（FoodScore）

## 概要
スマホ・PCのカメラで料理を撮影すると、Claude Vision APIが「色彩バランス・構図・明度・料理の配置」を分析して100点満点でスコアリングし、インスタ映え改善アドバイスを日本語で提示する撮影コーチアプリ。

## MVP機能
- カメラ起動 or 画像アップロード
- Claude APIで画像解析→採点（彩度・構図・明度・料理認識）
- 採点内訳の可視化（レーダーチャート）
- 「こう撮り直すと+15点」の具体的改善提案
- 過去の採点履歴と成長グラフ（localStorage）

## 技術スタック
- フロントエンド: Vanilla JS + Canvas API
- データ: localStorage（採点履歴）
- 特殊ライブラリ: Claude API Vision（claude-sonnet-4-6）、Chart.js（CDN）

## 難度
**高（1日フル）**

## 独自性
Instagram・TikTokのフィルター機能に似ているが、「撮影技術そのもの」をスコアリング・言語化してフィードバックする教育的UXが違う。フィルターで誤魔化すのではなく、腕を上げる。

## 20年後の視点
2045年のAI食文化では食事の見た目は栄養診断・健康管理AIの入力データとなる——視覚的食の記録習慣を今から作ることは、未来の健康データ資産の蓄積でもある。
