# キャストマイドック（CastMyDoc）

## 概要
文書やURLを貼ると、聞き手AIと解説AIの2人が対話するポッドキャスト台本に変換する学習ツール。日本語特化のNotebookLM風。

## MVP機能
- テキスト/URL入力→対話形式の台本生成（ボケ役の質問→解説役の回答）
- 台本の難易度切替（小学生向け/ビジネス向け）
- Web Speech APIでブラウザ読み上げ再生

## 技術スタック
- フロントエンド: Vanilla JS
- データ: localStorage（履歴）
- 特殊ライブラリ: Claude API / Web Speech API

## 難度
**中（半日）**

## 独自性
NotebookLMに似ているが、日本語の話芸（ボケとツッコミ）構造と難易度切替で「わかった気になれる」体験が違う

## 20年後の視点
2045年、学習コンテンツは「読む」から「対話を聞く」へ。その日本語版の実験。
