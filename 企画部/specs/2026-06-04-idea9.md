# 音声メモ帳 仕様書

## 概要
ブラウザのマイクで話した内容をWeb Speech APIでリアルタイムにテキスト化し、タグをつけて保存できるメモアプリ。「手を動かせない状況でもメモしたい」ニーズに対応。

## ターゲットユーザー
移動中・料理中・運転中など、手が離せない状況でメモしたい人

## MVP機能一覧
- [ ] マイクボタンで音声入力開始・停止
- [ ] Web Speech API（無料）でリアルタイムテキスト変換
- [ ] テキスト編集・修正が可能
- [ ] タグをつけて保存（仕事・アイデア・買い物など）
- [ ] Firebase Auth + Firestore で永続化
- [ ] タグでフィルタして一覧表示

## 将来拡張（MVP後）
- 音声ファイルも保存（Firebase Storage）
- メモをまとめてAIで要約
- リマインダー設定

## 技術スタック
- フロントエンド: React + Vite
- データ: Firebase Firestore + Auth
- ホスティング: Firebase Hosting
- 特殊ライブラリ: Web Speech API（ブラウザ標準）

## データモデル
```
voiceMemos/{uid}/memos/{memoId}
  - text: string
  - tags: string[]
  - createdAt: timestamp
```

## 画面構成
- ログイン画面
- メイン: 録音ボタン + テキストプレビュー + タグ入力 + 保存
- 一覧: タグフィルタ + メモカード一覧
