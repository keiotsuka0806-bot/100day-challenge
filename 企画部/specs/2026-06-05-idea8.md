# Retro Board（ふりかえりボード）

## 概要
チームや個人のふりかえりをオンラインで実施するツール。KPT（Keep/Problem/Try）形式でカードを貼り、アクションアイテムをまとめられる。URLシェアで即コラボ。

## MVP機能
- Keep / Problem / Try の3列構成
- カードを追加・編集・削除
- URL共有でリアルタイム共同編集（Firebase Realtime DB）
- カードに「いいね」ボタン（投票機能）
- アクションアイテムを別セクションに抽出

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Firebase Realtime Database
- 特殊ライブラリ: なし
