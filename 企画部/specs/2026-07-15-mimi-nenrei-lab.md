# 耳年齢ラボ（mimi-nenrei-lab）

## 概要
ブラウザだけで可聴域テスト（モスキート音）を行い「耳年齢」を推定。家族・友達と「あなたに聞こえてこの人に聞こえない音」を鳴らし合える。検査ではなく遊びとして設計。

## MVP機能
- Web Audioで8kHz→20kHzのスイープ再生→「聞こえなくなった点」で耳年齢推定
- 結果カード共有（耳年齢◯歳・聞こえた最高周波数）
- 「この音、聞こえる？」モード＝任意周波数を鳴らして家族と比べる

## 技術スタック
- Vanilla JS / Web Audio API / Canvas（完全オフライン・AI不使用・コストゼロ）

## 難度
**低（1〜2h）**

## 独自性
「聴力検査アプリに似ているが、医療でなく“世代の分断ネタ”として設計。親に聞こえず子に聞こえる音をその場で鳴らせる＝家族の集まりで盛り上がる形が違う」

## 収益の芽
なし（すごい/遊べる枠）。

## 20年後の視点
2045年、聴覚補正イヤホンが常識になり「生の耳年齢」は懐かしい遊びになる。

## 備考
端末スピーカーの高域特性で結果が揺れる＝「目安であり検査でない」明示が必須。医療誤認を避ける文言設計がゲート事項。
