# 冷蔵庫まるごと賞味期限（ReizoCal）

## 概要
冷蔵庫の中を1枚撮るとAI（Vision）が品目を読み取り、推定賞味期限つきで「使い切りカレンダー」に並べる。食品ロス防止に特化。

## MVP機能
- 冷蔵庫内の写真から品目を抽出
- 各品目の推定賞味期限を提示（手で訂正可）
- 期限が近い順のカレンダー/通知リスト

## 技術スタック
- フロントエンド: Vanilla JS / localStorage / PWA
- データ: ローカル
- 特殊: Vercel Function + OpenAI Vision（鍵なしはモック）
