# ペット記者会見（PetPresser）

## 概要
ペットの写真をアップすると、AIがそのペット「本人（本獣）」の記者会見コメントを生成し、会見フレーム付き画像にする大喜利ツール。

## MVP機能
- ペット写真アップロード
- AIが写真から性格を推定し「記者会見の発言」を数パターン生成
- マイク・社名ロゴ入りの会見フレームに合成して保存・シェア

## 技術スタック
- フロントエンド: Vanilla JS / Canvas
- データ: なし
- 特殊: Vercel Function + OpenAI Vision（鍵なしはモック）
