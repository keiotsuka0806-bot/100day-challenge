# PaletteGen 仕様書

## 概要
キーワードや気分から配色パレットを生成するツール。デザイナーや開発者が「雰囲気に合う色」を素早く見つけられる。

## ターゲットユーザー
Webデザイナー・フロントエンドエンジニア・個人開発者。配色に悩んでいる人。

## MVP機能一覧
- [ ] キーワード入力でパレット生成（ハードコードのマッピングで対応）
- [ ] 5色パレットを表示（HEX + RGB表示）
- [ ] 各色をクリックでHEXコードをコピー
- [ ] ランダム生成ボタン
- [ ] 色調の調整スライダー（明度・彩度）

## 将来拡張（MVP後）
- 画像からパレット抽出（Canvas API）
- パレットの保存・共有（Firebase）
- CSS変数として出力

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: キーワード→カラーセットのJSONマッピング（ハードコード）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし（HSL計算は自前）

## データモデル
```json
{
  "palettes": {
    "海": ["#1B4F72", "#2E86C1", "#85C1E9", "#D6EAF8", "#EBF5FB"],
    "森": ["#1D5E1D", "#27AE60", "#82E0AA", "#D5F5E3", "#F9FFF9"],
    "夕焼け": ["#922B21", "#E74C3C", "#F1948A", "#FADBD8", "#FFF5F5"]
  }
}
```

## 画面構成
1. **メイン** — キーワード入力 + ランダムボタン
2. **パレット表示** — 5色ボックス（HEX + コピーボタン）
3. **調整パネル** — 明度・彩度スライダー
