# 話速メーター（PaceTalk）

## 概要
プレゼンや面接の練習で「自分が今どれくらいの速さで話しているか」をマイクからリアルタイム可視化。速すぎ/ゆっくりをメーターと色で知らせ、終了後に平均話速・間の回数をレポート。

## MVP機能
- Web Speech API（or 音声区間検出）で発話を取得し1分あたり文字数を推定
- スピードメーター表示（速い=赤/適正=緑/遅い=青）＋リアルタイム警告
- 練習終了でサマリー（平均話速・最速区間・沈黙回数）

## 技術スタック
- フロントエンド: Vanilla JS / Canvas（メーター）
- データ: localStorage（練習履歴）
- 特殊ライブラリ: Web Speech API。APIなし=コストゼロ
