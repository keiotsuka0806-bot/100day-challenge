# TypeSpeed 仕様書

## 概要
1 分間のタイピング速度テスト。英語と日本語の両モード対応。WPM（1分あたりの単語数）とミス率でスキルを可視化。

## ターゲットユーザー
タイピングスキルを上げたい人。毎日ウォームアップとして使いたいエンジニア・ライター。

## MVP機能一覧
- [ ] 英語モード（ランダム英文フレーズ）
- [ ] 日本語モード（ランダム日本語文）
- [ ] 60 秒カウントダウン
- [ ] リアルタイム WPM 表示
- [ ] 正解した文字を緑、ミスを赤でハイライト
- [ ] 終了後スコア表示（WPM / 正確率）
- [ ] ベストスコアを LocalStorage に保存

## 将来拡張（MVP後）
- 難易度選択（かな / ひらがなのみ / 漢字混じり）
- スコアランキング（Firebase）
- コードタイピングモード

## 技術スタック
- フロントエンド: Vanilla JS
- データ: LocalStorage（ベストスコア）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```json
localStorage: {
  bestWPM_en: 85,
  bestWPM_ja: 120,
  bestAccuracy_en: 98.5
}
```

## 画面構成
- スタート: モード選択（英語 / 日本語）+ スタートボタン
- テスト中: 上部にテキスト表示、下部に入力欄 + タイマー + WPM
- 結果: スコアカード + ベスト更新時のお祝い表示

## デプロイ先
- Firebase project: type-speed-100day
- URL: https://type-speed-100day.web.app

## 難度
低（1〜2時間）
