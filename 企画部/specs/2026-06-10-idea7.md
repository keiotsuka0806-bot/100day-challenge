# MeetingCostMeter 仕様書

## 概要
会議の参加人数・役職・時間を入力するとリアルタイムで「この会議のコスト（円）」をカウントアップ表示するツール。「この会議、高すぎる」という気づきを与えるバイラルネタになる。

## リサーチ根拠
- 会議コスト可視化ツール系はTwitter/X等でたびたびバイラル（「時給換算で◯万円の会議」ネタ）
- 作るのは最もシンプルな部類（ロジック＋CSS演出のみ）
- 日本企業の「会議多すぎ問題」と「苦労キャンセル」（無駄排除）トレンドの両方に刺さる

## ターゲットユーザー
会議の多い日本企業のビジネスパーソン・エンジニア。Twitterでシェアして共感を集めたい人。

## MVP機能一覧
- [ ] 参加者設定（役職別・人数・想定時給）プリセット付き
- [ ] タイマースタートでリアルタイムコストカウントアップ（円）
- [ ] 「この会議で何が買えたか」対比表示（コーヒー◯杯・ランチ◯人分）
- [ ] スクリーンショット用の共有カード生成
- [ ] 過去の会議コスト履歴保存（localStorage）
- [ ] PWA対応

## 技術スタック
- フロントエンド: Vanilla JS
- データ: localStorage
- ホスティング: Firebase Hosting

## 画面構成
- 設定画面: 参加者ロール + 人数 + 会議時間（予定）
- カウンター画面: 巨大な金額表示 + 対比ユーモア表示
- 共有カード: 結果サマリーをPNG化してダウンロード / Xシェア

## デプロイ先
- Firebase project: 新規作成
- URL: https://meeting-cost-[id].web.app

## 難度・工数
低（1.5時間）
