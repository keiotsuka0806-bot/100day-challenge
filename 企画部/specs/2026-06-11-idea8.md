# ジェスチャーゲーム実験室（GestureGame）

## 概要
ブラウザカメラからMediaPipe Hand Landmarkで指ジェスチャーをリアルタイム検出し、「グーで攻撃・チョキで防御・パーでスキル」などの身体操作でゲームをプレイする実験的アクションゲーム。Physical AIトレンドへの入門体験。

## MVP機能
- カメラ映像から手のランドマーク21点をリアルタイム検出（MediaPipe CDN）
- ジャンケン3ポーズを認識してモンスター討伐ゲームに対応（グー=攻撃/チョキ=切断/パー=シールド）
- ゲームステータス（HP・スコア・ウェーブ数）をオーバーレイ表示
- ポーズ認識精度をデバッグモードで確認できる学習画面つき
- ハイスコアをlocalStorageに保存

## 技術スタック
- フロントエンド: Vanilla JS + Canvas API
- データ: localStorage（ハイスコア）
- 特殊ライブラリ: MediaPipe Tasks Vision（CDN）、@mediapipe/hands

## 難度
**中（半日）**

## 独自性
ジャンケンゲームは存在するが、**MediaPipeのランドマーク21点を使った精度の高いリアルタイムジェスチャーと本格的なゲームループ**を組み合わせた体験が異なる。身体とデジタルのインターフェース文法をゲームで習得できる。

## 20年後の視点
2045年にAR/VRグラスと空間コンピューティングが日常化した世界では、ジェスチャー操作が全UIの基本となる。このゲームは「空間OS以前に身体インターフェース文法を体験させる」予行演習だ。

## 備考
MediaPipe Tasks VisionはCDN配信あり、WebAssembly動作でサーバー不要。カメラ許可を得た後にSHORT_RANGE（近距離・高精度）モードで使用する。Webカメラ画質に依存するため、照明の良い環境推奨のUIガイダンスを表示する。
