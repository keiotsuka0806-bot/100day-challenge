# ローカルメモリベース 仕様書

## 概要
プロジェクトの作業ログ・意思決定メモ・学びをブラウザに貼り続けると、Claude APIが「なぜこの決定をしたか」「何を学んだか」を自動で構造化・タグ付けして保存するローカルナレッジベース。Product Hunt週間2位「Minimi」のローカル個人版コンセプト。

## リサーチ根拠
- Minimi（2026年6月PH週間2位, スコア520）: 「AIがコミュニケーションを監視してコンテキスト記憶」ツールが急需要
- ただしMinimiはクラウド・企業向け → 個人開発者のローカル版ニーズは未充足
- 既存のDailyQuite・AICompanion（会話型）・VoiceDiaryとは完全に異なる「プロジェクト意思決定の構造化記録」

## ターゲットユーザー
個人開発者・フリーランス。「なぜあの時こう決めたか」が後で思い出せなくて困っている人。

## MVP機能一覧
- [ ] テキストエリアに作業ログ・メモを自由に貼り付け
- [ ] Claude APIが自動で「決定事項 / 学び / ToDo / 疑問」に分類・タグ付け
- [ ] 構造化されたメモカードを時系列で一覧表示
- [ ] キーワード検索（全文）
- [ ] タグフィルター（決定 / 学び / ToDo / 疑問）
- [ ] エクスポート（JSON / Markdown）
- [ ] すべてlocalStorageに保存（クラウド不要）
- [ ] PWA対応

## 技術スタック
- フロントエンド: Vanilla JS
- API: Claude API（Vercel Serverless Function経由）
- データ: localStorage
- ホスティング: Vercel

## 画面構成
- 入力エリア: テキストペースト + AIで整理ボタン
- カード一覧: タグ別グループ + 時系列
- 検索バー + タグフィルター
