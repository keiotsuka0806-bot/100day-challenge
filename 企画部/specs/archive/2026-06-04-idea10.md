# ミニ家計簿 仕様書

## 概要
シンプルな収支記録アプリ。レシートを見ながら手入力するだけで、カテゴリ別の支出グラフと月次サマリーを自動生成する。銀行連携なしでも使える軽量家計簿。

## ターゲットユーザー
複雑な家計簿アプリが面倒で続かなかった人、シンプルに支出を把握したい人

## MVP機能一覧
- [ ] 収支の入力（金額・カテゴリ・メモ・日付）
- [ ] カテゴリ選択（食費・交通費・娯楽・その他など）
- [ ] 今月の支出合計 + カテゴリ別円グラフ
- [ ] 直近の入力履歴一覧（削除可能）
- [ ] Firebase Auth + Firestore で永続化

## 将来拡張（MVP後）
- 月をまたいだ履歴・前月比較
- 予算設定と超過アラート
- CSVエクスポート

## 技術スタック
- フロントエンド: React + Vite
- データ: Firebase Firestore + Auth
- ホスティング: Firebase Hosting
- 特殊ライブラリ: Chart.js（CDN）

## データモデル
```
expenses/{uid}/records/{recordId}
  - amount: number
  - category: string
  - memo: string
  - date: string (YYYY-MM-DD)
  - type: 'expense' | 'income'
  - createdAt: timestamp
```

## 画面構成
- ログイン画面
- メイン: 今月サマリー + 円グラフ + 入力フォーム
- 履歴: 日付順リスト + 削除
