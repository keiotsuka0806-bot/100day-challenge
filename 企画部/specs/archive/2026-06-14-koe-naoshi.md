# 声から整文（KoeNaoshi）

## 概要
ブラウザの音声認識で喋った内容をその場で文字起こしし、Claude APIが「です・ます調の整文＋箇条書き化」だけを行う。会話せず、整えるだけ。散歩中や移動中の思いつきを下書きに変える。

## MVP機能
- ブラウザ音声認識(Web Speech API)でリアルタイム文字起こし
- 「整文だけ」「箇条書き化」「note下書き化」モードを選択
- Claude APIで体裁を整えて出力→コピー

## 技術スタック
- フロントエンド: Vanilla JS / Web Speech API
- データ: なし(セッション内のみ)
- 特殊ライブラリ: Claude API(整文用)

## 難度
**中（半日）**

## 独自性
「ChatGPTの音声入力」に似ているが、あれは会話になりがちで質問を返してくる。これは**返事をせず整文だけして黙る**。Keiが朝の散歩で浮かんだnoteのネタを、その場で整った下書きにできる。

## 20年後の視点
2045年、入力は声が主流。だが「会話ではなく成形だけしてほしい」需要は残り続ける。

## 備考
Claude APIキーは当面Kei自身用(BYOK禁の学びより、公開時はサーバ側鍵＋利用制限を検討)。トレンドの音声/エッジAIに合致。
