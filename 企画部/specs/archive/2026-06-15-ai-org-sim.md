# AI組織シミュレータ（AIOrgSim）

## 概要
AI組織を画面上で組み立て、部署(ノード)と情報の流れ(矢印)を編集すると、構造変化の影響(情報フロー・ボトルネック・副作用)をシミュレートするツール。「組織構造そのものを触れる」体験。

## MVP機能
- 5部署を初期ノードとして組織図に表示／矢印に流れる情報名を表示
- 部署の追加・削除・接続編集(ドラッグ&ドロップ含む)
- 「この組織で1日をシミュレート」→ 全体変化/良くなる点/悪化点/ボトルネック/各部署の反応/改善提案
- AI未設定でも動くモックシミュレーション(グラフ構造から決定的に分析)

## 技術スタック
- React / Vite / TypeScript / React Flow(@xyflow/react)。AI分析(任意)は Vercel Function + OpenAI API。
- ※方針転換の経緯: 当初「組織変更を寸劇で出すアプリ(Vanilla+OpenAI)」→ 2026-06-15に「組織図を編集して構造影響をシミュレートするツール(React Flow)」へ作り直し。複雑UIのためReact採用。
