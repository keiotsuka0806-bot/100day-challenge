# WordFlash 仕様書

## 概要
英単語をフラッシュカード形式で覚えるアプリ。「覚えた / まだ」で振り分けながら繰り返し学習できる。

## ターゲットユーザー
英語学習中の社会人・学生。スキマ時間にスマホで単語を復習したい人。

## MVP機能一覧
- [ ] 単語リストを JSON で管理（初期データ 50 語）
- [ ] カードをタップでフリップ（表: 英語 → 裏: 日本語 + 例文）
- [ ] 「覚えた」「まだ」ボタンで振り分け
- [ ] 「まだ」カードだけ再チャレンジモード
- [ ] セッション結果を表示（正答率）
- [ ] LocalStorage で「覚えた」単語を記憶

## 将来拡張（MVP後）
- カスタム単語リスト追加
- スペルチェックモード（入力）
- 学習履歴グラフ

## 技術スタック
- フロントエンド: Vanilla JS
- データ: JSON（静的ファイル）+ LocalStorage
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし（CSS flip アニメーションで実装）

## データモデル
```json
words.json: [
  { "en": "ephemeral", "ja": "はかない・一時的な", "example": "Youth is ephemeral." }
]
localStorage: { masteredIds: ["word-0", "word-5", ...] }
```

## 画面構成
- スタート画面: 「全単語モード / 未習得のみ」選択
- フラッシュカード画面: 1枚ずつ表示、フリップ + 判定ボタン
- 結果画面: 今日のスコア + 累計習得数

## デプロイ先

- URL: https://word-flash-100day.web.app
