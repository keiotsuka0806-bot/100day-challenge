# MeetingAction 仕様書

## 概要
会議のメモや議事録テキストを貼り付けるだけで、Claude APIが「アクションアイテム・担当者・期限」のみを抽出してToDo形式で出力するツール。Granola・Mina Meeting Assistantのシンプル版。

## ターゲットユーザー
会議後に議事録をまとめるのが面倒な社会人・チームリーダー。英語・日本語混在のメモでも動く。

## MVP機能一覧
- [ ] テキストエリアに会議メモを貼り付け
- [ ] 「抽出する」ボタンでClaude APIに送信
- [ ] アクションアイテム一覧をチェックボックス形式で表示（担当者・期限付き）
- [ ] 「Markdownでコピー」「プレーンテキストでコピー」ボタン
- [ ] Vercel Serverless FunctionでAPIキー隠蔽

## 将来拡張（MVP後）
- Slack / Notionへのワンクリックエクスポート
- 過去の抽出履歴をFirestoreに保存
- マイク録音→リアルタイム文字起こし統合

## 技術スタック
- フロントエンド: Vanilla JS
- バックエンド: Vercel Serverless Function（Claude API）
- ホスティング: Vercel
- 特殊ライブラリ: なし

## データモデル
- ステートレス（保存なし）
- Serverless Functionがリクエストごとに Claude API を呼ぶ

## 画面構成
- 上: テキストエリア（会議メモ入力）
- 中: 「抽出する」ボタン（ローディングアニメあり）
- 下: アクションアイテム一覧 + コピーボタン

## デプロイ先
- URL: https://meeting-action.vercel.app（仮）
