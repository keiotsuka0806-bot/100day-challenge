# VibeSanitizer 仕様書

## 概要
Cursor・Copilot・Lovableなど「Vibe coding」で生成したコードをペーストするとClaude APIがAPIキー露出・認証漏れ・XSS・SQLインジェクションなどのセキュリティ問題を瞬時スキャンするツール。一般的なコードレビューではなく「AI生成コード特有のリスク」に特化。

## リサーチ根拠
- Product Hunt / Indie Hackers 2026年調査で「vibe code security scanning」が急成長ニッチとして明示
- "tools that check AI-generated code for common security mistakes" の需要が急増
- CodeReviewBotは一般コードレビュー、本ツールはAI生成コード特化で完全に異なるニーズ

## ターゲットユーザー
Vibe codingで爆速開発しているが、セキュリティが心配なソロ開発者・スタートアップ。

## MVP機能一覧
- [ ] コード貼り付けエリア（言語自動検出）
- [ ] Claude APIによるスキャン（5カテゴリ: APIキー露出 / 認証バイパス / XSS / SQLi / 危険なeval）
- [ ] 各リスクの重大度（Critical/High/Medium）+ 該当行番号 + 修正提案
- [ ] 「安全度スコア」サマリー表示（0-100）
- [ ] スキャン結果のコピー / URLシェア
- [ ] PWA対応

## 技術スタック
- フロントエンド: Vanilla JS
- API: Claude API（Vercel Serverless Function経由）
- ホスティング: Vercel

## 画面構成
- 左: コード入力エリア + スキャンボタン
- 右: リスク一覧カード（重大度バッジ付き）+ 安全度スコアメーター

## デプロイ先
