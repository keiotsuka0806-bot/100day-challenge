# 個人開発ダッシュボード 仕様書

## 概要
自分が作ったアプリのURL・説明・ステータスを一元管理するポートフォリオ兼ダッシュボード。「何を作ったか」を整理して外部シェアもできる。（トレンド反映: 個人開発者の「届ける」課題に対応）

## ターゲットユーザー
複数の個人開発プロジェクトを抱える開発者

## MVP機能一覧
- [ ] プロジェクト登録（名前・URL・説明・ステータス・技術スタック）
- [ ] カード形式でプロジェクト一覧表示
- [ ] ステータス管理（開発中 / 稼働中 / 停止中）
- [ ] 公開URLでポートフォリオとしてシェア可能
- [ ] Firebase Auth + Firestore で永続化

## 将来拡張（MVP後）
- PV数・いいね数などの指標入力
- タグ・技術スタックでフィルタ
- OGP画像の自動生成

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Firebase Firestore + Auth
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```
projects/{uid}/items/{projectId}
  - name: string
  - url: string
  - description: string
  - status: 'active' | 'wip' | 'archived'
  - stack: string[]
  - createdAt: timestamp
```

## 画面構成
- ログイン画面
- ダッシュボード: プロジェクトカード一覧 + 追加ボタン
- 公開ページ: 読み取り専用カード一覧
