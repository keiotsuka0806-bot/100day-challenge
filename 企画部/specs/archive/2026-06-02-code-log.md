# CodeLog 仕様書

## 概要
OSSや業務コードを読む際の気づきや疑問をその場でメモするアプリ。ファイル名・行番号付きで記録でき、後から検索・整理しやすい。

## ターゲットユーザー
コードリーディングを習慣にしているエンジニア。読んだコードの理解を記録として残したい人。

## MVP機能一覧
- [ ] メモ記録（ファイル名・行番号・メモ本文）
- [ ] タグ付け（疑問・気づき・TODO・重要の4種）
- [ ] プロジェクト名でグループ化
- [ ] タグ・プロジェクトでフィルター
- [ ] Googleログイン（Firebase Auth）

## 将来拡張（MVP後）
- GitHubリポジトリとの連携
- コードスニペット埋め込み
- チームでの共有機能

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: Firebase Firestore
- 認証: Firebase Auth (Google Sign-In)
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
Firestoreコレクション: `codeLogs/{uid}/entries/{docId}`
```json
{
  "project": "react/react",
  "file": "packages/react/src/ReactHooks.js",
  "line": 42,
  "memo": "useStateの実装を確認。dispatchはqueueに積まれる",
  "tag": "気づき",
  "createdAt": "timestamp"
}
```

## 画面構成
1. **ログイン画面**
2. **ホーム** — 最近のメモ一覧 + フィルター
3. **新規メモ入力** — フォーム（プロジェクト・ファイル・行番号・メモ・タグ）
4. **プロジェクト一覧** — グループ別集計
