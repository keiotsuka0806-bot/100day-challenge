# 音だけ神経衰弱（Oto Memory）

## 概要
カードに絵はなく、めくると「音」だけが鳴る神経衰弱。同じ音のペアを記憶だけで当てる。目をつぶってもできる、聴覚版の定番ゲーム。

## MVP機能
- 6〜12枚の伏せカード、タップで音が鳴る（鳥・水滴・ベル・笑い声など）
- 2枚めくって同じ音ならペア成立、違えば伏せに戻る
- 手数/タイムでスコア、2人で交互の対戦モード

## 技術スタック
- フロントエンド: Vanilla JS / Web Audio API / Canvas
- データ: localStorage（ベストスコア）
- 特殊: 音は合成 or 短いサンプル。AI不要・コストゼロ

## 難度
**低（1〜2h）**

## 独自性
「絵柄の神経衰弱に似ているが、"音だけ"で記憶することで、視覚に頼れず新鮮な難しさが生まれ、目の不自由な人とも一緒に遊べる」点が違う（Keiの多様性への関心に重なる）。

## 20年後の視点
2045年、視覚過多の世界で"音だけで遊ぶ"設計はアクセシビリティと新鮮さの両面で意味を持ち続ける。
