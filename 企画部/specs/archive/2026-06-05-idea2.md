# Habit Chain（習慣チェーン）

## 概要
毎日の習慣を「チェーン」として可視化するアプリ。連続達成日数が伸びるほどチェーンが長くなり、「絶対に切りたくない」モチベーションを作る。

## MVP機能
- 習慣を複数登録（名前・絵文字アイコン）
- 毎日「達成」ボタンで連続日数カウントアップ
- チェーン（○が繋がる帯グラフ）で進捗を可視化
- 過去カレンダービューで達成日を色付き表示
- 現在の連続記録・最長記録を表示

## 技術スタック
- フロントエンド: Vanilla JS + CSS Grid
- データ: Firebase Firestore
- 特殊ライブラリ: なし

## 難度
**低（1〜2h）**

## 備考
継続習慣系は熱狂的なファンがつきやすい。MoodForecastとの連携拡張も可能。
