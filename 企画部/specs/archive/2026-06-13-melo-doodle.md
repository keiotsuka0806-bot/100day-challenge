# メロディスケッチ（MeloDoodle）

## 概要
画面のグリッドに音を置くだけで短いメロディを作り、その場で再生してWAV/MIDIで書き出せる。鼻歌の代わりに作曲の"とっかかり"を残し、Sunoなどへのインスピレーション素材にする。

## MVP機能
- ピアノロール風グリッドでの音の配置・再生（WebAudio）
- テンポ・スケール(メジャー/マイナー/ペンタ)の選択
- WAV書き出し（最低限）と、配置の保存（localStorage）

## 技術スタック
- フロントエンド: vanilla JS + Web Audio API
- データ: localStorage
- 特殊ライブラリ: WAVエンコードの軽量ロジック（自前 or 小型ライブラリ）

## 難度
**高（1日フル）**

## 独自性
「本格DAWに似ているが、"鼻歌レベルの思いつきを30秒で形にして書き出す"とっかかり特化で、機能の多さを捨てた点が違う」

## 20年後の視点
2045年、曲の完成はAIがやる。だが「最初のひらめき(モチーフ)」を人が置く瞬間は創作の起点として残る。種をまく道具の価値は消えない。
