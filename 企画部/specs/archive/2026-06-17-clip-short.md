# ブラウザ録画→字幕焼き込みショート（ClipShort）

## 概要
画面/カメラをブラウザで録画し、音声から自動字幕を付けて焼き込み、縦型ショート動画として書き出すツール。すべてローカル完結。

## MVP機能
- 画面 or カメラ録画（MediaRecorder）
- Web Speech API で音声をテキスト化し、タイムライン字幕として編集
- FFmpeg.wasm で字幕焼き込み＋9:16クロップして mp4 出力

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし（端末内処理）
- 特殊ライブラリ: MediaRecorder / Web Speech API / FFmpeg.wasm

## 独自性
「CapCut等を開かず、録って字幕付けて縦型で出すまでがブラウザ1枚で完結」。Keiが開発実況のショートを作る時に使える。

## 20年後の視点
2045年、撮影＝編集済み。字幕やクロップを後から人がやる工程は消える。

## 独自性の弱点
ShortsStudio（CapCutとの違い不明で休止）と同じ轍を踏むリスク。FFmpeg.wasmは重く1日に収まりにくい。
