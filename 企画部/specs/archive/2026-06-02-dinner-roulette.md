# 今日なに食べる？仕様書

## 概要
「今日の晩ごはん何にしよう」という毎日の悩みを解決するルーレットアプリ。条件を選ぶとスロットマシン風アニメーションで1品が提案される。

## ターゲットユーザー
一人暮らしや共働き夫婦で、毎日の献立に悩む人。

## MVP機能一覧
- [ ] カテゴリ選択（和食・洋食・中華・なんでも）
- [ ] 調理時間フィルター（15分以内 / 30分以内 / なんでも）
- [ ] スロットアニメーションで料理名を表示
- [ ] 料理名・主材料・一言コメントを表示
- [ ] 「もう一回」ボタンで再抽選

## 将来拡張（MVP後）
- 「今日は作った！」記録機能
- 食材入力で逆引き検索
- Rakuten Recipe API連携

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: 料理JSONファイル（50〜80件をハードコード）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```json
{
  "meals": [
    {
      "name": "親子丼",
      "category": "和食",
      "time": 15,
      "ingredients": ["鶏肉", "卵", "玉ねぎ"],
      "comment": "家にある材料で簡単に作れる定番"
    }
  ]
}
```

## 画面構成
1. **トップ** — カテゴリ・時間フィルター選択
2. **結果** — スロットアニメーション → 料理カード表示
3. （モーダル）詳細情報（材料・コメント）
