# 傾き玉転がし2P対戦（Tilt Maze 2P）

## 概要
スマホ2台で対戦。各自が自分の端末を物理的に傾けて、迷路の中の玉をゴールへ運ぶ速さを競う。同じ部屋でQR参加してリアルタイムに相手の進捗が見える。

## MVP機能
- 部屋作成→QR/コードで2台目が参加（Firestore同期）
- 各端末で傾き（DeviceOrientation）操作、玉が迷路を転がる
- 相手の現在位置/進捗バーをリアルタイム表示、先にゴールで勝ち
- 3本勝負・リマッチ

## 技術スタック
- フロントエンド: Vanilla JS / Canvas
- データ: Firestore（共有倉庫 tatsuro-kei-2026・匿名auth）でマッチ同期
- 特殊: DeviceOrientation API / PWA

## 独自性
「ひとり用の玉転がしに似ているが、"2台を傾けてその場で competing する身体対戦"にすることで、画面でなく相手の手元と歓声で盛り上がる」点が違う。Face Magic Battleで作った2台対戦の資産を活かせる。

## 20年後の視点
2045年も、同じ場で身体を使って競う遊びの熱量はオンライン越しでは代替できない。
