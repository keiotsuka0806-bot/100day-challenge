# しゃべって日報3形式（DiaryTalk）

## 概要
1日の出来事を箇条書き(または音声)で入れると、Claude APIが「日報・note素材・自分用振り返り」の3形式に同時展開する。Keiの日次運用(3フォルダ更新)の素材出しを1発で済ませる。

## MVP機能
- 箇条書き/音声入力で「今日やったこと」を投入
- 3タブ出力: ①事実ベースの日報 ②感情・物語を足したnote素材 ③学び抽出(記憶庫形式)
- 各タブをコピー、Markdownで保存

## 技術スタック
- フロントエンド: Vanilla JS / (任意)Web Speech API
- データ: localStorage(下書き)
- 特殊ライブラリ: Claude API

## 難度
**高（1日フル）**

## 独自性
「ChatGPTに今日のことを書かせる」に似ているが、毎回「日報用に」「note用に」と3回指示し直す必要がある。これは**1回の入力で運用に必要な3形式が同時に出る**。Keiの日報・note・記憶庫の更新を毎晩一本化できる。

## 20年後の視点
2045年、ライフログは自動収集される。だが「同じ素材を文脈別に出し分ける」編集需要は人間の発信が続く限り残る。

## 備考
