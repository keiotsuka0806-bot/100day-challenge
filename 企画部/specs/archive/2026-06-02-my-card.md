# MyCard 仕様書

## 概要
自己紹介プロフィールカードを作成してURLで共有できるサービス。SNSリンク・スキルタグ・一言コメントをオシャレなカードとして公開できる。

## ターゲットユーザー
エンジニア・フリーランサー・就活生。名刺代わりのリンクを作りたい人。

## MVP機能一覧
- [ ] プロフィール作成（名前・肩書き・一言コメント・アバター画像）
- [ ] SNSリンク追加（Twitter/X・GitHub・LinkedIn）
- [ ] スキルタグ追加（最大10個）
- [ ] ユニークURLで公開（/[username]）
- [ ] Googleログイン（Firebase Auth）

## 将来拡張（MVP後）
- カードデザイン選択（テーマ複数）
- 閲覧数カウント
- QRコード表示

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: Firebase Firestore
- 認証: Firebase Auth (Google Sign-In)
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
Firestoreコレクション: `profiles/{username}`
```json
{
  "uid": "firebase_uid",
  "name": "山田 太郎",
  "title": "フロントエンドエンジニア",
  "bio": "Webを作るのが好きです",
  "avatar": "data:image/jpeg;base64,...",
  "links": {
    "twitter": "https://x.com/example",
    "github": "https://github.com/example"
  },
  "skills": ["JavaScript", "React", "Firebase"],
  "createdAt": "timestamp"
}
```

## 画面構成
1. **ログイン画面**
2. **編集画面** — フォーム入力 + プレビュー表示
3. **公開カード** — `/[username]` でアクセス（ログイン不要）

## デプロイ先
- Firebase project: 新規作成（my-card-app）
- URL: https://my-card-app.web.app
