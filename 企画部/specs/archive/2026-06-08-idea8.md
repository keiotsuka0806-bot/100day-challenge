# SkillSwap 仕様書

## 概要
「英語教えます↔デザイン教えて」のようなスキル交換・物々交換の掲示板アプリ。お金を介さずスキルでつながるコミュニティ。Community skill-sharing の成長トレンドに乗る。

## ターゲットユーザー
お金より時間やスキルで価値交換したいフリーランサー・学生・副業者。地域・オンライン問わずスキルシェアの相手を探している人。

## MVP機能一覧
- [ ] Firebase Auth（Google）でログイン
- [ ] 投稿機能: 「提供できるスキル」「欲しいスキル」「ひとこと」「オンライン/対面」
- [ ] 投稿一覧（Firestoreリアルタイム）
- [ ] スキルタグフィルター（デザイン / 言語 / エンジニアリング / 料理 / 音楽 / etc.）
- [ ] 「興味あり」ボタン → 投稿者のFirestoreに通知フラグ立て
- [ ] 自分の投稿管理ページ（編集・削除）

## 将来拡張（MVP後）
- ダイレクトメッセージ機能
- マッチング成立後のレビュー
- 位置情報フィルター（近くのスキルスワップ）

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Firebase Firestore + Firebase Auth
- ホスティング: Firebase Hosting
- 特殊ライブラリ: Firebase SDK（CDN）

## データモデル
```
posts/{postId}:
  uid, displayName, offerSkill, wantSkill, message, mode, tags[], createdAt, interestedCount

users/{uid}/interests/{postId}:
  timestamp
```

## 画面構成
- トップ: 投稿一覧（タグフィルター付き）
- モーダル: 新規投稿フォーム
- マイページ: 自分の投稿一覧
