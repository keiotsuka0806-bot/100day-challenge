# ワードチェーン対戦「WordBattle」仕様書

## 概要
しりとりをAI（ルールベース）と対戦するゲーム。制限時間内に答えられないと負け。日本語しりとりに特化したシンプルなブラウザゲーム。

## ターゲットユーザー
暇つぶしにゲームを楽しみたい人。語彙力を鍛えたい人。

## MVP機能一覧
- [ ] プレイヤー vs CPU のしりとり対戦
- [ ] 10秒の制限時間（カウントダウン表示）
- [ ] 使用済み単語の管理（重複防止）
- [ ] 「ん」で終わったら負け判定
- [ ] ゲームオーバー画面 + スコア（何回続いたか）

## 将来拡張（MVP後）
- 2人対戦モード（同一端末）
- 難易度選択（CPUの応答速度）
- ランキング（Firebase）

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: 単語辞書JSONファイル（3000語程度）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```json
{
  "words": ["りんご", "ごりら", "らっぱ", "ぱいなっぷる"]
}
```
ゲーム中状態はメモリ上のみ（localStorage不要）

## 画面構成
1. **タイトル画面** — スタートボタン
2. **ゲーム画面** — 最後の単語表示 + 入力フォーム + タイマー + 使用済み単語リスト
3. **ゲームオーバー** — スコア表示 + リトライボタン
