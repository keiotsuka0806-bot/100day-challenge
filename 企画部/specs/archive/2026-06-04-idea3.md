# ひとこと日記 仕様書

## 概要
毎日1行だけ書く、超シンプルな日記アプリ。長文を書く必要がないので続けやすい。過去の記録をカレンダーで振り返れる。

## ターゲットユーザー
日記を続けたいが続かない人、忙しくても記録をつけたい人

## MVP機能一覧
- [ ] 今日の一言を入力して保存（1日1件）
- [ ] 過去の記録をカレンダービューで表示
- [ ] 記録がある日はカレンダーにドットを表示
- [ ] 日付をタップして過去のひとことを確認
- [ ] Firebase Auth（Googleログイン）でデバイスをまたいで同期

## 将来拡張（MVP後）
- 気分タグ（😊😐😔）を付与
- 連続記録日数のストリーク表示
- 月ごとのエクスポート

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Firebase Firestore + Auth
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```
diaryEntries/{uid}/entries/{YYYY-MM-DD}
  - text: string
  - createdAt: timestamp
```

- ログイン画面
- メイン: 今日の入力欄 + カレンダー
