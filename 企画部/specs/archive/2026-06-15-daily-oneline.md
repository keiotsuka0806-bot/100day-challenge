# 日報の一行化（DailyOneLine）

## 概要
一日の出来事をだらだら貼ると、「明日の自分が3秒で思い出せる1行」に圧縮する。

## MVP機能
- 当日メモを貼り付け
- Claudeが象徴的な1行に圧縮
- 過去の1行をカレンダー風に一覧

## 技術スタック
- Vanilla JS / localStorage / Claude API

## 難度
**低（1〜2h）**

## 独自性
「DiaryTalk（しゃべって日報3形式）に似ているが、出力を“1行だけ”に絞る点が違う」

## 20年後の視点
2045年、記録は長さを競わず、想起のしやすさで価値が決まる。

## 備考
DiaryTalkと出力目的が近く、被り懸念あり。
