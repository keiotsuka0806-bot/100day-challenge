# ContentRePurpose 仕様書

## 概要
ブログ記事・ツイート・メモを貼り付けるとClaude APIがX（Twitter）・Instagram・LinkedIn・Threads向けに最適化したテキストへ自動リライトするツール。1つの素材から4プラットフォーム分の投稿を一気に生成。

## ターゲットユーザー
コンテンツを複数SNSで発信している個人開発者・ライター・マーケター。書いたあとの「SNS展開が面倒」問題を解決したい人。

## MVP機能一覧
- [ ] テキストエリアに元コンテンツを入力
- [ ] プラットフォーム選択（X / Instagram / LinkedIn / Threads / 全部）
- [ ] 「リライト」でClaude APIが各プラットフォームの制限・文化・トーンに合わせて生成
  - X: 140字以内・ハッシュタグ付き・カジュアル
  - Instagram: 絵文字込み・ストーリー性・改行あり
  - LinkedIn: プロフェッショナルトーン・学び・CTA付き
  - Threads: Xに近いが少し長め・フレンドリー
- [ ] 各プラットフォームタブで結果表示 + 個別コピー
- [ ] Vercel Serverless FunctionでAPIキー隠蔽

## 将来拡張（MVP後）
- 「再生成」でトーン違いのバリエーション追加
- 予約投稿（Buffer / Hootsuite API連携）
- 画像キャプション生成（Claude Vision）

## 技術スタック
- フロントエンド: Vanilla JS
- バックエンド: Vercel Serverless Function（Claude API）
- ホスティング: Vercel
- 特殊ライブラリ: なし

## データモデル
- ステートレス（保存なし）
- localStorage で最後の入力テキストのみキャッシュ

## 画面構成
- 上: テキストエリア（元コンテンツ）+ プラットフォーム選択チェックボックス
- 中: 「リライト」ボタン
- 下: タブ切替で各プラットフォームの生成結果 + コピーボタン
