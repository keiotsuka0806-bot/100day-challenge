# README Generator 仕様書

## 概要
リポジトリ名・説明・使用技術・主な機能を入力するだけで、Claude APIがGitHubに映える美しいREADME.mdを自動生成するツール。バッジ・インストール手順・使い方・スクリーンショットプレースホルダーまで含む。

## ターゲットユーザー
README を書くのが億劫な個人開発者・学生。OSS公開前の最後の一押しに悩んでいる人。

## MVP機能一覧
- [ ] 入力フォーム: プロジェクト名 / 概要 / 技術スタック / 主要機能（箇条書き）/ デモURL / ライセンス
- [ ] 「生成する」でClaude APIが Markdown形式のREADMEを出力
- [ ] プレビュー（Marked.jsでリアルタイムMarkdownレンダリング）
- [ ] 「Markdownコピー」ワンクリック
- [ ] Vercel Serverless FunctionでAPIキー隠蔽

## 将来拡張（MVP後）
- GitHubリポジトリURLを入力→コードを読んで自動README生成
- テンプレート選択（シンプル / 詳細 / OSS向け）
- 複数言語対応（EN / JP）

## 技術スタック
- フロントエンド: Vanilla JS + Marked.js（CDN）
- バックエンド: Vercel Serverless Function（Claude API）
- ホスティング: Vercel
- 特殊ライブラリ: Marked.js（Markdownレンダリング）

## データモデル
- ステートレス（保存なし）

## 画面構成
- 左: 入力フォーム
- 右: Markdownプレビュー
- フッター: コピーボタン

## デプロイ先
- URL: https://readme-gen.vercel.app（仮）
