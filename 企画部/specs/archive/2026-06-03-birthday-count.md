# BirthdayCount 仕様書

## 概要
大切な人の誕生日までのカウントダウンを表示するシンプルなアプリ。毎日起動するたびに「あと〇日」が励みになる。

## ターゲットユーザー
家族・友人・パートナーの誕生日を大切にしたい人。スマホのホーム画面に置いて毎日確認したい人。

## MVP機能一覧
- [ ] 誕生日リストを登録（名前 + 日付）
- [ ] 今日から誕生日まで「あと〇日」を表示
- [ ] 今日が誕生日の人を特別表示（🎂 + メッセージ）
- [ ] LocalStorage に永続化
- [ ] PWA 対応（ホーム画面に追加）

## 将来拡張（MVP後）
- プッシュ通知（誕生日の朝に通知）
- 年齢の表示
- メッセージテンプレート共有

## 技術スタック
- フロントエンド: Vanilla JS
- データ: LocalStorage
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```json
birthdays: [
  { id: "uuid", name: "田中さん", month: 3, day: 15 }
]
```

## 画面構成
- メイン画面: カード形式でカウントダウン一覧（近い順にソート）
- 追加モーダル: 名前 + 月日の入力フォーム

## デプロイ先
- Firebase project: birthday-count-100day
- URL: https://birthday-count-100day.web.app
