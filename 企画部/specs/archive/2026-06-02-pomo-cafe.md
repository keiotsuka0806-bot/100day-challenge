# PomoCafe 仕様書

## 概要
カフェの環境音とポモドーロタイマーを組み合わせた集中支援アプリ。雨音・カフェ・焚き火の音が流れる中で25分集中できる。

## ターゲットユーザー
在宅勤務や自宅学習で集中力が続かない人。カフェの雰囲気が好きだが外出が面倒な人。

## MVP機能一覧
- [ ] 25分 / 5分のポモドーロサイクル
- [ ] BGM選択（雨音・カフェ・焚き火の3種）
- [ ] 残り時間の大きな円形表示
- [ ] 終了時のブラウザ通知（Notification API）
- [ ] BGM音量調整スライダー

## 将来拡張（MVP後）
- カスタム時間設定
- 完了回数の記録
- Spotify連携

## 技術スタック
- フロントエンド: Vanilla JS + CDN
- データ: localStorage（設定保存のみ）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: Web Audio API（ループ再生）
- 音源: Freesound.orgのCC0音源 or CDN配信

## データモデル
localStorage キー: `pomo_cafe_settings`
```json
{
  "bgm": "rain",
  "volume": 0.6,
  "workMin": 25,
  "breakMin": 5
}
```

## 画面構成
1. **メイン** — タイマー円 + BGM選択 + スタート/ストップボタン
2. （ポップアップ）セッション完了通知 + 次のセッション開始確認
