# 心拍ドラム（PulseDrum）

## 概要
カメラに顔を映すだけで、肌の微小な色変化（rPPG）から心拍を推定し、その鼓動をビートにして曲を生成する。ウェアラブル不要で「自分の心臓が音楽になる」体験。

## MVP機能
- カメラで顔の頬領域の緑チャンネル平均を時系列で取得→バンドパス＋FFTで心拍数(BPM)推定
- 推定したBPMをテンポにしてWeb Audioでドラム＋ベースを自動生成
- 心拍に合わせて脈打つビジュアル＋BPM表示。30秒ループを書き出し

## 技術スタック
- フロントエンド: Vanilla JS / Web Audio API / Canvas
- データ: なし（端末内のみ）
- 特殊ライブラリ: 軽量FFT（自前 or fft.js）

## 難度
**高（1日フル）**

## 独自性
心拍計アプリは「数値を出す」で終わる。これは“自分の鼓動そのものが曲のテンポになる”ので、測定が表現に変わる。カメラだけで完結する点も驚き。

## 20年後の視点
2045年、生体データは常時音楽やUIに溶け込む。その源流として「カメラ一つで心臓を聴いた」小品は、身体とメディアが混ざり始めた瞬間の記録になる。

## 備考
rPPGはブラウザ/照明条件で精度が不安定＝1日で“全力で完成”できるか要検証（精度が出ない場合は「ゆらぎ演出」に振る逃げ道を用意）。医療目的でない注記必須。
