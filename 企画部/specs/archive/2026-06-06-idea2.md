# ColorExtract — 写真から支配色パレットを抽出するツール

## 概要
画像をアップロードするとCanvas APIで画素を解析し、支配的な5色のカラーパレットを生成する。HEXコードをワンクリックでコピーできる。

## MVP機能リスト
1. 画像アップロード（ドラッグ＆ドロップ対応）
2. Canvas APIで全画素をサンプリングし、k-means的なクラスタリングで5色抽出
3. 各色のHEXコードを表示しクリックでコピー
4. 生成したパレットをPNGとしてダウンロード
5. 直近3件のパレットをLocalStorageで保持

## 技術スタック
- フロントエンド: Vanilla JS + Canvas API
- データ: LocalStorage
- 特殊ライブラリ: なし（Canvas + 独自クラスタリング実装）

## 難度
**低（1–2h）**
k-meansは簡易版（メジアンカット法など）で十分。ビジュアルのインパクトが大きく、SNSで映える。
