# PWAアイコン一括ジェネレータ（IconForge）

## 概要
1枚の正方形画像を入れると、PWAに必要な全サイズ（192/512/maskable/apple-touch）と manifest.json の icons 断片を、ブラウザ内で一括生成してzipでDLできるツール。

## MVP機能
- 画像ドロップ → Canvasで各サイズへリサイズ＆余白（safe area）付きmaskable生成
- manifest.json の icons 配列スニペットをコピー可能な形で出力
- 全アイコン＋manifest断片をzipでまとめてDL

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし（完全ローカル・画像はアップロードしない）
- 特殊ライブラリ: Canvas API ＋ JSZip（CDN）

## 独自性
「既存のオンラインアイコン生成は画像アップロードが必要で広告だらけ。これはローカル完結＆このスタジオのmanifest形式に最適化済み」。Keiは毎日PWAアプリ（本社ルールで原則PWA化）を作るので、今日の新プロジェクトのアイコン準備で必ず使う。
