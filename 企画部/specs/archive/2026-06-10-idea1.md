# RetroSnap 仕様書

## 概要
写真をアップするとCCD調ノイズ・色収差・日付スタンプを加えてレトロデジカメ風に変換するフィルターツール。日経トレンディ2026年予測「レトロ調デジカメブーム」の波に乗るシェアネタアプリ。

## リサーチ根拠
- 日経トレンディ2026年ヒット予測で「レトロ調デジカメ」がムード消費トレンドとして明示
- Instagram/Xで「フィルムカメラ風」写真は2025年から継続的にバイラル
- Canvas APIのみで外部依存ゼロ → 1日で完成する

## ターゲットユーザー
スマホ写真をレトロに加工してSNSにシェアしたい10〜30代。

## MVP機能一覧
- [ ] 画像アップロード（ドラッグ&ドロップ / タップ）
- [ ] フィルタープリセット3種（CCD 2000年代 / フィルム1990年代 / ポラロイド風）
- [ ] 各フィルターの強度スライダー
- [ ] 日付スタンプ（年月日 / オレンジ文字）のON/OFF
- [ ] PNG / JPEG でダウンロード
- [ ] PWA対応

## 技術スタック
- フロントエンド: Vanilla JS + Canvas API
- データ: なし（クライアントのみ）
- ホスティング: Firebase Hosting

## 画面構成
- アップロードエリア（ドロップゾーン）
- プレビューキャンバス
- フィルタープリセット選択 + 強度スライダー
- ダウンロードボタン

## デプロイ先
- Firebase project: 新規作成
- URL: https://retro-snap-[id].web.app
