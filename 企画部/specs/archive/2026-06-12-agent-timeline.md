# エージェント社史メーカー（AgentTimeline）

## 概要
GitリポジトリのURLを入れると、コミット履歴からプロジェクトの歩みを「会社の社史」風タイムラインページに自動生成するツール。ビルド・イン・パブリック支援。

## MVP機能
- GitHubリポジトリURL→コミット履歴取得（GitHub API）
- AIが節目（初デプロイ・大改修・組織変更）を抽出して章立て
- 縦スクロールのタイムラインページを生成・公開URL化

## 技術スタック
- フロントエンド: Vanilla JS
- データ: GitHub API / Firebase Hosting
- 特殊ライブラリ: Claude API

## 難度
**高（1日フル）**

## 独自性
GitHubのグラフ表示に似ているが、数字でなく「物語」として節目を語らせる点が違う
