# TypeDuel — AIゴーストと戦うリアルタイムタイピングゲーム

## 概要
お題の文章をタイピングしながら、自分の過去の最速記録（ゴースト）とリアルタイムで競うタイピングゲーム。WPMを日別に記録して成長曲線を可視化する。

## MVP機能リスト
1. ランダムなお題文章（日本語・英語）を表示し、タイピングを開始
2. 現在のWPM・正確率・経過時間をリアルタイム表示
3. 前回の自己ベストをゴーストカーソルとして同期表示（進行速度の視覚化）
4. 完了後にWPM・正確率・ハイスコアを記録（LocalStorage）
5. 日別WPMの折れ線グラフで成長曲線を表示（Chart.js）

## 技術スタック
- フロントエンド: Vanilla JS + CSS アニメーション
- データ: LocalStorage
- 特殊ライブラリ: Chart.js CDN（成長曲線グラフ）
