# あいさつ発音コーチ（Aisatsu Coach）

## 概要
海外旅行前に、現地語の基本あいさつ（こんにちは/ありがとう/いくら？）をマイクに向かって言うと、AIが発音を聞いて「通じる度」を採点し、どの音を直せばいいか平易にコーチする。

## MVP機能
- 国/言語を選ぶ→頻出あいさつ10個のお手本（テキスト＋読み上げ）
- マイク録音→AIが発音評価（通じる度％＋直すポイント1つ）
- 合格したフレーズにスタンプ／旅行前カードとして一覧化

## 技術スタック
- フロントエンド: Vanilla JS / Web Audio API / Web Speech（お手本読み上げ）
- データ: localStorage（進捗）
- 特殊: Vercel Function + OpenAI（音声→発音講評。鍵なしは録音再生＋自己採点でモック）

## 難度
**中（半日）**

## 独自性
「翻訳・語学アプリに似ているが、"旅行で実際に使う10フレーズだけ"を発音に絞って即コーチするので、文法学習せず明日の旅で通じる準備が15分で済む」点が違う。Keiの海外経験とも親和。

## 20年後の視点
2045年、同時通訳が普及しても「自分の口で一言伝わる」喜びは残り、その練習場としての価値は続く。

## 備考
発音評価は厳密でなく"通じる/励ます"トーンで。トレンド（AI音声）を実用に転用した1案。
