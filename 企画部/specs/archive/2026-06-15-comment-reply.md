# コメント返信下書き（CommentReply）

## 概要
note/SNSで受け取ったコメントを貼ると、自分のトーンを保ったまま返信案を3つ出す。

## MVP機能
- コメント貼り付け＋自分のトーン指定
- 返信案を3つ生成
- トーン（丁寧/フランク/熱め）切替

## 技術スタック
- Vanilla JS / localStorage / Claude API

## 難度
**低（1〜2h）**

## 独自性
「一般的な返信生成に似ているが、“自分の過去文からトーンを学ぶ”点が違う」

## 20年後の視点
2045年、人格を保った自動返信が当たり前になり、本人が書く返信は贈り物になる。

## 備考
Claude API案。ただしKeiはまだコメントが多くなく、今週使う場面が立てづらい。
