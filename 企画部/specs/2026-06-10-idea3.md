# ありがとうカード 仕様書

## 概要
感謝の気持ちをアニメーション付きインタラクティブカードにしてURLでシェアできるツール。受け取った相手はブラウザで開くだけで演出が再生される。日本の「お礼文化」に特化したeカードサービス。

## リサーチ根拠
- 2026年2月、solo devがインタラクティブeカードサービスを7日で構築し$10,000を売り上げた事例が話題（Medium記事）
- 日本には誕生日・卒業・入社・退職など「感謝を伝える場面」が欧米より多い文化
- URLシェアのみで動く → Firebase不要、サーバーレス（カード内容をURLパラメータにエンコード）

## ターゲットユーザー
退職・誕生日・卒業などで感謝を伝えたい日本のユーザー全般。

## MVP機能一覧
- [ ] テンプレート選択（退職 / 誕生日 / 卒業 / 入社 / 感謝）
- [ ] メッセージ・差出人名の入力
- [ ] カードアニメーション3種（紙吹雪 / 桜 / 星）
- [ ] カードをURLエンコードしてシェアURL生成
- [ ] 受取人はURLを開くだけでアニメーション再生
- [ ] 画像としてダウンロード（Canvas → PNG）
- [ ] PWA対応

## 技術スタック
- フロントエンド: Vanilla JS + Canvas API
- データ: URLパラメータ（base64エンコード）
- ホスティング: Firebase Hosting

## 画面構成
- 作成画面: テンプレート + テキスト入力 + アニメーション選択 + URLコピー
- 受取画面: URLパラメータを読んでカードアニメーション再生

## デプロイ先
- Firebase project: 新規作成
- URL: https://arigato-card-[id].web.app
