# 名言法廷 仕様書

## 概要
「その名言、本当にその人が言ったのか？」を裁判形式で判決するゲーム。
有名な名言の多くが実は誤引用・創作であるという歴史的事実を題材に、
本物 or フェイクを見極める推理×学習ゲーム。

## ターゲットユーザー
歴史・偉人の言葉に興味がある人。「アインシュタインの名言」を鵜呑みにしてきた人。

## MVP機能
- 10の名言事件（本物5 / フェイク5）
- 起訴状（疑惑のヒント）を読んで「本物」か「フェイク」を判決
- 判決後に真実と解説を表示
- 最終スコア＋結果サマリー
- localStorage でハイスコア保存

## 技術スタック
- Vanilla JS / HTML / CSS（ビルドステップなし）
- PWA対応（manifest.json + service worker）
- Firebase Hosting でデプロイ

## 将来拡張
- Claude API で「弁護側・検察側の主張」を動的生成
- 事件データベースの拡張（50件以上）
- タイムアタックモード
- SNSシェア機能
