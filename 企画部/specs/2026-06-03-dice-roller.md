# DiceRoller 仕様書

## 概要
TRPG・ボードゲーム向けのサイコロロールアプリ。複数種類のサイコロを組み合わせてロール、履歴も残せる。

## ターゲットユーザー
TRPG（D&D・クトゥルフなど）プレイヤー。ボードゲームでスマホをサイコロ代わりに使いたい人。

## MVP機能一覧
- [ ] サイコロ種類: d4 / d6 / d8 / d10 / d12 / d20 / d100
- [ ] 各サイコロの個数指定（1〜10）
- [ ] ロールボタンでアニメーション付き結果表示
- [ ] 合計値 + 各ダイスの内訳を表示
- [ ] 直近 10 件の履歴を表示
- [ ] シェイク操作でもロール（DeviceMotion API）

## 将来拡張（MVP後）
- カスタム修正値（+2 など）
- プリセット保存（「攻撃ロール: 1d20+5」など）
- 複数プレイヤーモード

## 技術スタック
- フロントエンド: Vanilla JS
- データ: LocalStorage（履歴）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし（CSS アニメーションで演出）

## データモデル
```json
rollHistory: [
  { timestamp: "2026-06-03T21:00:00", dice: "2d20", results: [14, 7], total: 21 }
]
```

## 画面構成
- メイン: サイコロ選択パネル + 大きな「ROLL」ボタン
- 結果: アニメーション後に合計値をドーンと表示、内訳をリスト
- 履歴: 画面下部にスクロール

## デプロイ先
- Firebase project: dice-roller-100day
- URL: https://dice-roller-100day.web.app
