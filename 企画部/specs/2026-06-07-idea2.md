# CLIコマンドタイピング道場（CommandDojo）

## 概要
git・docker・bashの頻出コマンドをタイピング練習で身につける開発者向けゲーム。正確なコマンド入力習慣を楽しく形成する。

## MVP機能
- git / docker / bash カテゴリ選択
- コマンドが上から流れてきてタイプするアーケード形式
- スコア・連続正解数をlocalStorageに保存してハイスコア表示

## 技術スタック
- フロントエンド: Vanilla JS
- データ: localStorage（スコア・ハイスコア）
- 特殊ライブラリ: なし

## 難度
**低（1〜2h）**

## 独自性
TypeKanaに似ているが、対象を「開発者が毎日使うCLIコマンド」に絞った実用ゲーム。練習と業務効率向上が直結する。

## 20年後の視点
AIがCLIを代行する2045年でも、コマンドの意味を理解する「人間側のリテラシー」は希少価値として残る。

## 備考
コマンド選定リストをJSONで外部管理すれば、k8s・awscli等に拡張しやすい。開発者コミュニティへのシェアで拡散が見込める。
