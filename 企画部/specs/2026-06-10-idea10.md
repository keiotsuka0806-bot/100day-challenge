# タイムラプスコード（TimelapseCode）

## 概要
GitHubリポジトリのURLを入力するとコミット履歴を取得し、ファイル追加・変更・削除の波をアニメーションで再生する「コードの進化タイムライン」ビジュアライザー。

## MVP機能
- GitHubリポジトリURL入力 → GitHub API でコミット履歴取得
- コミット時系列バー：縦軸=ファイル数、横軸=時間、色=追加(緑)/削除(赤)/変更(黄)
- 「再生」ボタンでコミット単位のアニメーション（Canvas描画）
- コミットカードのホバーで差分サマリー・コミットメッセージ表示
- 再生速度スライダー（1x / 5x / 20x）

## 技術スタック
- フロントエンド: Vanilla JS
- データ: GitHub REST API（公開リポジトリのみ・レート制限考慮）
- 特殊ライブラリ: なし（Canvas APIで自前アニメーション）

## 難度
**高（1日フル）**

## 独自性
gource（ローカルCLIツール）に似ているが、「URLを貼るだけでブラウザで即再生・インストール不要」という即時性と、モバイルでも共有できる点が違う。

## 20年後の視点
2045年にはコードをAIが自律生成するため「人間がコードを書いた時代の記録アーカイブ」として文化的・歴史的価値を持ち、このツールがその観賞体験を提供するだろう。

## 備考
GitHub Trending急上昇プロジェクトの多くがAIコーディングツールであることから「コードの進化を見る」需要は高い。gourceはインストール必須で敷居が高く、ブラウザ版の空白地帯を狙える。デモとしてこのリポジトリ自身を入力するという「メタ的展示」が可能。
