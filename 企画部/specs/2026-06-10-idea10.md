# MoodPlaylist 仕様書

## 概要
今の気分・状況をタップするとClaude APIがSpotify / YouTube向けの具体的な楽曲リストを提案するムードベース音楽ディスカバリーアプリ。日本2026年トレンド「ムード消費」を直撃。

## リサーチ根拠
- 日経トレンディ2026年予測「ムード消費」: ユーザーが「今この瞬間の気分」でコンテンツを選ぶ傾向が強まる
- Spotifyの「Daily Mix」「Mood Playlists」機能は人気だが「今このモードで集中したい」という粒度に対応できていない
- PomoCafe（Pomodoro×環境音）・Virtual Study Roomとは異なり「音楽発見・SpotifyURL提案」に特化

## ターゲットユーザー
気分に合わせた音楽を素早く見つけたい10〜30代。Spotifyユーザー。

## MVP機能一覧
- [ ] ムードカード選択（集中 / リラックス / 元気を出したい / 落ち込み回復 / 作業BGM / 眠れない）
- [ ] 状況タグ追加（雨の日 / 深夜 / 朝 / 運動 / 読書 など）
- [ ] Claude APIが10曲のプレイリストを提案（曲名 + アーティスト + 選定理由1行）
- [ ] Spotify / YouTube で検索するリンクを自動生成
- [ ] お気に入りプレイリスト保存（localStorage）
- [ ] PWA対応

## 技術スタック
- フロントエンド: Vanilla JS
- API: Claude API（Vercel Serverless Function経由）
- データ: localStorage
- ホスティング: Vercel

## 画面構成
- ムード選択画面: カードグリッド（6〜8パターン）
- 状況タグ選択（任意）
- 結果画面: 10曲リスト + Spotify/YouTubeリンク

## デプロイ先
- Vercel project: mood-playlist
- URL: https://mood-playlist.vercel.app（仮）

## 難度・工数
高（1日フル）— Spotify / YouTubeリンク生成ロジックと、Claude APIのプロンプト最適化に時間がかかる
