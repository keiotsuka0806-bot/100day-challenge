# 仕様書: VinRoute（ヴァンルート）— ワイン産地ボードゲーム

**作成日**: 2026-06-03  
**ステータス**: 企画中  
**想定工数**: 5日（Phase 1〜3）

---

## コンセプト

ワイン産地を旅しながら畑を買い、ブドウを収穫し、ワインを醸造して「最高のソムリエ」を目指すボードゲーム。桃鉄の「移動・産地探訪」とカタンの「リソース管理・建設・交渉」を融合し、ソムリエ知識（産地・品種・テイスティング）がゲームの戦略に直結する設計。

---

## 建設・開発（カタン要素）

| 建物 | コスト | 効果 |
|------|--------|------|
| 畑（Vineyard） | ブドウ×2 | 毎ターン通過時にリソース+1 |
| 醸造所（Winery） | ブドウ×3 + 金×1 | ワインを醸造可能に |
| セラー（Cellar） | 全リソース×1 | 熟成ボーナス（翌ターン価値2倍）|
| ワインバー（Wine Bar） | ワイン×2 | 他プレイヤーとの交渉でリソース+1 |

---

## ワインレシピ（勝利点）

醸造所を持つプレイヤーが特定リソースを消費してワインを「醸造」→勝利点を獲得。

| ワイン名 | 必要リソース | 勝利点 |
|---------|------------|--------|
| ボルドー・ブレンド | カベルネ×2 + ブレンド×1 | 3点 |
| ブルゴーニュ・ルージュ | ピノ×3 | 4点（単一品種ボーナス）|
| シャンパーニュ | シャルドネ×2 + 金×1 | 3点 |
| スパークリング・ロゼ | ピノ×1 + シャルドネ×1 | 2点 |
| グランクリュ（特級） | 全品種×1 | 6点 |
| 甲州ワイン | リースリング×1 + ブレンド×2 | 2点（+テイスティング解放）|

---

## 移動システム（桃鉄要素）

- 毎ターン**サイコロ2個**を振り、出た目の数だけ産地ルートを移動
- 止まった産地でそのエリアのリソースを取得
- **特急カード**（一気に遠くへ移動）
- **目的地ボーナス**: ターン開始時にランダムで「今月の銘醸地」が表示され、到達したプレイヤーに+2点

---

## イベントカード（桃鉄要素）

ターン終了時に1枚引く。

| カード名 | 効果 |
|---------|------|
| 豊作（Bonne Récolte）| 自分の畑のリソース+2 |
| 霜害（Gel Printanier）| 選択した相手の畑リソース-1 |
| 著名評論家（Parker登場）| 最高点ワインを持つプレイヤーに+1点 |
| ワインスキャンダル | 全プレイヤーの「ブレンド」リソースを半分に |
| 気候変動 | ランダムで1エリアの産出リソース種が変わる |
| 貿易風（桃鉄ボンビー相当）| 最下位プレイヤーに付き、通過した産地の畑に1ターンダメージ |

---

## ソムリエクイズシステム

「テイスティングカード」を引いたターン、または山梨エリアで止まったとき発動。

- クイズに**正解** → 特殊アクション（リソース盗む・追加移動・勝利点+1）
- クイズに**不正解** → なにも起きない（ペナルティなし、参加しやすい）

### クイズ例
- 「ボルドー5大シャトーを2つ答えよ」
- 「ピノ・ノワールの主要産地は？」
- 「シャブリはどの品種で作られる？」
- 「スパークリングワインの製法"瓶内二次発酵"を何と呼ぶ？」
