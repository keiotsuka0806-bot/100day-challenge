# SoundBoard — ブラウザで音を録音して並べるカスタムサウンドパッド

## 概要
マイクで録音した音やシステム音をボタンに割り当て、クリックで即再生できるサウンドパッド。配信・実況・作業BGM用途に使える。

## MVP機能リスト
1. マイクから音声を録音してバッファに保存（MediaRecorder API）
2. 12個のパッドボタンに録音音声をアサイン
3. パッドクリックで即時再生（Web Audio API）
4. ループ再生モードのON/OFFトグル
5. パッド名称の編集とIndexedDBへの保存（再訪問でも設定維持）

## 技術スタック
- フロントエンド: Vanilla JS + CSS Grid（パッド配置）
- データ: IndexedDB（音声バイナリ保存）
- 特殊ライブラリ: Web Audio API + MediaRecorder API（ブラウザ標準）
