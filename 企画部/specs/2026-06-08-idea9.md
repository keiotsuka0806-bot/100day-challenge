# WorkflowBuilder 仕様書

## 概要
ドラッグ&ドロップでAI処理ステップを繋ぐビジュアルワークフローエディタ。「テキスト入力→要約→翻訳→SNS投稿文生成」のようなパイプラインをノードで組み立て、実行できるブラウザアプリ。n8nのシンプル版。

## ターゲットユーザー
コードを書かずにAIワークフローを組みたい非エンジニア。プロンプトチェーンを視覚的に設計したい個人開発者。

## MVP機能一覧
- [ ] ノード種別: Input（テキスト入力）/ Prompt（Claude API呼び出し）/ Output（結果表示）
- [ ] SVGキャンバス上でノードをドラッグ&ドロップ配置
- [ ] ノード間を線でつないでデータフロー定義
- [ ] 各Promptノードにシステムプロンプトと入力変数を設定
- [ ] 「実行」ボタンでInputからOutputまで順次実行
- [ ] Vercel Serverless FunctionでClaude API呼び出し
- [ ] ワークフローのJSON export / import

## 将来拡張（MVP後）
- 条件分岐ノード（if/else）
- HTTP Requestノード（外部API接続）
- Firestoreへの保存・共有

## 技術スタック
- フロントエンド: Vanilla JS + SVG（ライブラリなしでキャンバス描画）
- バックエンド: Vercel Serverless Function（Claude API）
- ホスティング: Vercel
- 特殊ライブラリ: なし（SVGを手書き）

## データモデル
```json
{
  "nodes": [
    { "id": "1", "type": "input", "x": 100, "y": 200, "value": "..." },
    { "id": "2", "type": "prompt", "x": 300, "y": 200, "systemPrompt": "...", "userPrompt": "{{input}}" },
    { "id": "3", "type": "output", "x": 500, "y": 200 }
  ],
  "edges": [
    { "from": "1", "to": "2" },
    { "from": "2", "to": "3" }
  ]
}
```

## 画面構成
- 左サイドバー: ノードパレット（ドラッグ元）
- メイン: SVGキャンバス（ノード配置・接続）
- 右パネル: 選択中ノードの設定フォーム
- フッター: 実行ボタン + ログ表示エリア
