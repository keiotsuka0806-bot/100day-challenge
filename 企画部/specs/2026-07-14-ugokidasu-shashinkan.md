# 動きだす写真館（UgokidasuShashinkan）

## 概要
手持ちの写真1枚をブラウザにドロップすると、端末内の深度推定で前景と背景を分離し、視差でふわっと立体に動く「生きてる写真」になる。ループ動画として書き出して送れる。古い家族写真が動く瞬間の驚き。

## MVP機能
- 写真ドロップ→端末内深度推定（ONNX Runtime Web + Depth Anything small）→2.5D視差アニメーション
- カメラワーク3種（寄る・回る・こちらへ）とループ動画(WebM/GIF)書き出し
- すべて端末内処理（写真をどこにも送らない=プライバシー訴求）

## 技術スタック
- Vanilla JS / ONNX Runtime Web / WebGL / Canvas
- AI API不使用（モデルはCDN配信・推論は端末内）=ランニングコストゼロ

## 難度
**高（1日フル）**

## 独自性
「iPhoneの空間写真・Googleフォトのシネマティックフォトに似ているが、機種を選ばず・古いデジカメ写真やスキャンした紙焼きでも・ブラウザだけで動く点が違う」

## 収益の芽
なし（すごい枠。強いてC型: 法事・記念日向け高解像度書き出し）

## 20年後の視点
2045年には平面写真は自動で空間化されて閲覧されるのが標準で、これはその過渡期の手品として見える。
