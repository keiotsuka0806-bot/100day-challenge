# 声で描く光の絵（VoicePaint）

## 概要
マイクに向かって声を出すと、音程が色に、音量が線の太さに変換され、光の線がリアルタイムに伸びていくジェネラティブ・アート。録音して作品を保存できる。

## MVP機能
- マイク入力の音程（ピッチ）・音量をリアルタイム解析
- 音程→色相、音量→線幅・発光量で軌跡を描画（残光・ブルーム）
- 録った1枚をPNG/短いGIFで書き出し・シェア

## 技術スタック
- フロントエンド: Vanilla JS / Web Audio API / Canvas（合成・発光）
- データ: なし
- 特殊: なし（AI不要・コストゼロ）

## 20年後の視点
2045年、表現が自動生成で溢れる世界で、これは「自分の身体（声）が一次素材であること」をまだ大切にしている小さな手仕事。
