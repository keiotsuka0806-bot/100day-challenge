# バーコード仮想本棚（BarcodeShelf）

## 概要
持っている本のバーコード（ISBN）をカメラで読むと、書名を取得して背表紙が仮想本棚に並んでいく。積読も含めて"自分の蔵書"を眺める楽しさと、棚をシェアして本好き同士でつながる実用＋収集アプリ。

## MVP機能
- カメラでISBNバーコード読取（BarcodeDetector）→Open Library等で書名取得
- 背表紙を自動生成して本棚に追加（localStorage保存）
- 棚の並べ替え・"読了/積読"タグ・棚画像のシェア

## 技術スタック
- フロントエンド: Vanilla JS / Canvas（背表紙描画）
- データ: localStorage
- 特殊ライブラリ: BarcodeDetector API / Open Library API（無料）。AI課金なし

## 難度
**中（半日）**

## 独自性
「読書管理アプリに似ているが、登録の手間をバーコード一発にし、"見て嬉しい本棚ビジュアル"に振り切った点が違う」

## 20年後の視点
2045年、紙の本はコレクター品に。物理の蔵書をデジタルの棚に写し取る習慣は、本好きのアイデンティティ表現になっているかも。

## 備考
書名取得の精度（和書のヒット率）が要検証。BarcodeDetector非対応ブラウザはZXing CDNでフォールバック。
