# AIコード絵文字コメンター（EmojiDoc）

## 概要
コードスニペットをペーストするとClaude APIが各行・各ブロックに適切な絵文字と日本語コメントを自動付与し、「感情的に読めるコード」を生成するWebツール。

## MVP機能
- コード貼り付け → Claude APIで絵文字+日本語コメント自動付与
- 言語自動判定（JS / Python / Go / Ruby等）
- コメントトーン選択（ユーモラス・真面目・関西弁）
- ワンクリックでクリップボードコピー

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし
- 特殊ライブラリ: highlight.js（CDN）、Claude API（claude-haiku-4-5）

## 難度
**低（1〜2h）**

## 独自性
GitHub CopilotやCodeWhisperに似ているが、「コードに感情・絵文字・関西弁を混ぜて読みやすくする」という真剣ではないユーティリティとして差別化。チーム内レビューのアイスブレイクにも使える。

## 20年後の視点
2045年のペアプログラミングはAIと感情的文脈を共有しながら進む——コードに感情的注釈を加えるという概念は「共感コーディング」のプロトタイプ。
