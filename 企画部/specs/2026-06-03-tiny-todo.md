# TinyTodo 仕様書

## 概要
「今日やること」を 3 つだけ登録できる超シンプルな TODO アプリ。4 つ目を追加しようとするとブロックされる制約が本質。

## ターゲットユーザー
タスクが多すぎて何も進まない人。「今日は本当に大事な 3 つだけ」に集中したい人。

## MVP機能一覧
- [ ] タスクを最大 3 つまで登録
- [ ] 4 つ目を入力しようとすると「今日はこの 3 つに集中！」とブロック
- [ ] タスクをチェックで完了
- [ ] 日付が変わると自動でリセット（今日のタスクは今日だけ）
- [ ] LocalStorage で当日データを保持
- [ ] PWA 対応

## 将来拡張（MVP後）
- 完了タスクの日次履歴
- 翌日への繰り越し機能（あえて 1 つだけ許可）

## 技術スタック
- フロントエンド: Vanilla JS
- データ: LocalStorage（日付キーで管理）
- ホスティング: Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
```json
localStorage key: "tinytodo-2026-06-03"
value: [
  { id: "1", text: "企画書を書く", done: false },
  { id: "2", text: "PR を出す", done: true }
]
```

## 画面構成
- 1画面のみ: 大きなテキスト入力 + 3つのカードスロット
- 満杯時: スロットが「ロック」ビジュアルに変化

## デプロイ先
- Firebase project: tiny-todo-100day
- URL: https://tiny-todo-100day.web.app

低（1〜2時間）
