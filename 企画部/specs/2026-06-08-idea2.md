# BentoProfile 仕様書

## 概要
Bento Gridレイアウトで自分の自己紹介カードを作り、URLでシェアできるプロフィールジェネレーター。名前・肩書き・SNSリンク・得意なこと・好きなものを入力するだけでスタイリッシュなプロフィールページが完成する。

## ターゲットユーザー
ポートフォリオを手軽に持ちたいエンジニア・フリーランサー・学生。Notionやリンクまとめ系ツールの代替を探している人。

## MVP機能一覧
- [ ] フォームで名前・肩書き・ひとこと・SNSリンク（X/GitHub/LinkedIn）・スキルタグを入力
- [ ] Bento Gridレイアウトでリアルタイムプレビュー表示
- [ ] URLパラメータにデータをエンコードしてシェア可能（Firebase不要）
- [ ] ライトモード / ダークモード切替
- [ ] 「URLをコピー」ワンクリック

## 将来拡張（MVP後）
- Firebase保存でカスタムスラッグURL（/profile/kei）
- カラーテーマ選択
- プロフィール画像アップロード
- OGP画像自動生成

## 技術スタック
- フロントエンド: Vanilla JS + CSS Grid
- データ: URLパラメータ（Base64エンコード）
- ホスティング: Vercel or Firebase Hosting
- 特殊ライブラリ: なし

## データモデル
- URLクエリパラメータに JSON をBase64エンコードして持たせる
- `?data=eyJuYW1lIjoi...` 形式

## 画面構成
- 左: 入力フォーム
- 右: Bento Gridプレビュー（リアルタイム更新）
- フッター: URLコピーボタン + シェアボタン

## デプロイ先
- URL: https://bento-profile.vercel.app（仮）

## 難度・工数
低（2時間）
