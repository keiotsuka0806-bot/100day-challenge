# GroupTimeline 仕様書

## 概要
招待コードで繋がった少人数グループが同じタイムラインに投稿できる共同日記アプリ。家族・友人・チームで「今日あったこと」を気軽に共有するプライベートSNS。

## ターゲットユーザー
遠距離の家族・友人グループ・チームで日常を共有したい人。Instagramより気軽でプライベートな場が欲しい人。

## MVP機能一覧
- [ ] Firebase Auth（Google）でログイン
- [ ] グループ作成 → 招待コード生成（6桁）
- [ ] 招待コードでグループ参加
- [ ] テキスト投稿（写真は将来拡張）をFirestoreリアルタイムで全員に反映
- [ ] 投稿に「いいね」絵文字リアクション（❤️ 👍 😂 😮）
- [ ] タイムライン表示（新しい順）
- [ ] 自分の投稿のみ削除可能

## 将来拡張（MVP後）
- 写真投稿（Firebase Storage）
- コメント機能
- プッシュ通知（新規投稿時）
- グループ設定（名前・アイコン変更）

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Firebase Firestore + Firebase Auth
- ホスティング: Firebase Hosting
- 特殊ライブラリ: Firebase SDK（CDN）

## データモデル
```
groups/{groupId}:
  name, inviteCode, members: [uid], createdAt

groups/{groupId}/posts/{postId}:
  uid, displayName, text, reactions: { "❤️": [uid], "👍": [uid] }, createdAt
```

## 画面構成
- ログイン画面: Googleサインイン
- ホーム: グループ一覧 or グループ参加/作成
- タイムライン: 投稿一覧 + 投稿フォーム（フローティングボタン）
- 各投稿: アバター + 名前 + 本文 + リアクション + 時刻
