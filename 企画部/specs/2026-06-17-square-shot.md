# スクショ整形（SquareShot）

## 概要
スクショを貼ると、余白を自動検出してトリミングし、指定比率（1:1 / 16:9 / 4:5）に背景色付きでフィットさせてDLできるツール。SNS投稿用の画像を一発で整える。

## MVP機能
- 画像ドロップ → 周囲の単色余白を自動検出してクロップ
- 出力比率と背景色（白/黒/カスタム）を選んでフィット配置
- 複数枚をまとめて同じ設定で書き出し

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし（ローカル完結）
- 特殊ライブラリ: Canvas API

## 難度
**低（1〜2h）**

## 独自性
「プレビュー.appで手作業トリミングするより、貼って比率を選ぶだけで複数枚を一括整形」。KeiがnoteのアイキャッチやXの画像を整える時に使える。

## 20年後の視点
2045年、投稿先に合わせた画像整形は撮影時に自動完結。後工程の手トリミングは消える。

## 備考
余白の自動検出（角のピクセル色を基準に許容差で塗りつぶし範囲を判定）が実装の肝。
