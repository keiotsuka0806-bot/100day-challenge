# AI俳句批評家（HaikuSensei）

## 概要
ユーザーが作った俳句をClaude APIに送ると、松尾芭蕉・正岡子規・小林一茶の三視点から批評・改善提案・類句を返してくれる学習＆創作支援ツール。

## MVP機能
- 俳句入力フォーム（5-7-5カウント補助）
- 三俳人スタイルの批評（各200字以内）
- 「季語あり/なし」「写生/心情」等の自動分類タグ付け
- 過去作一覧と批評の保存（localStorage）
- AIによる類似名句サジェスト

## 技術スタック
- フロントエンド: Vanilla JS
- データ: localStorage
- 特殊ライブラリ: Claude API（claude-sonnet-4-6）

## 難度
**中（半日）**

## 独自性
俳句投稿サービス（角川俳句等）に似ているが、AIが歴史的俳人の批評スタイルを模倣して即座にフィードバックする点が違う。人間の選者を待たなくていい。

## 20年後の視点
2045年のAI生成コンテンツ氾濫の時代に、人間が5-7-5の制約の中で季語と着眼点を選ぶ俳句こそが「人間性の証明」として再評価される——その創作力を今から鍛える意義がある。

## 備考
Claude API のシステムプロンプトで「芭蕉として回答してください」と役割設定する。日本文化・伝統芸術のデジタル化という文化的価値もある。
