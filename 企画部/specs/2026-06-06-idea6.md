# HabitStreak — 習慣トラッカーと炎ストリーク可視化

## 概要
毎日こなしたい習慣を登録し、チェックを続けると炎アイコンのストリーク数が伸びていくアプリ。継続のモチベーションを視覚化する。

## MVP機能リスト
1. 習慣の登録・削除（最大10個）
2. 今日の完了チェック（チェックすると炎が増える）
3. 各習慣の現在ストリーク数と最長ストリーク数を表示
4. 週間カレンダーで過去7日間のチェック履歴をヒートマップ表示
5. Firebase Firestoreでデータ保持（Google認証でマルチデバイス対応）

## 技術スタック
- フロントエンド: Vanilla JS + CSS Grid
- データ: Firebase Firestore + Firebase Auth（Google認証）
- 特殊ライブラリ: なし

## 難度
**中（半日）**
Firestoreのデータ設計（習慣×日付の記録構造）とストリーク計算ロジックに時間がかかる。認証フローは過去のパターン流用可。
