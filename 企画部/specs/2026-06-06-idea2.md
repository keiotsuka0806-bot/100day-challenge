# テキスト縦書きビジュアライザー（TatePrint）

## 概要
入力したテキストをリアルタイムに縦書きプレビューし、PNG画像としてダウンロードできるWebツール。年賀状・祝辞・短歌・詩の縦書きレイアウトを即座に作成。

## MVP機能
- テキスト入力→即座に縦書きCSS表示
- フォント・文字サイズ・背景色・文字色のカスタマイズ
- Canvas API経由でPNG出力（A4比率 / 正方形 / SNS正方形）
- ルビ（振り仮名）対応の簡易入力記法

## 技術スタック
- フロントエンド: Vanilla JS
- データ: なし（ステートレス）
- 特殊ライブラリ: なし（CSS writing-mode + Canvas API のみ）

## 難度
**低（1〜2h）**

## 独自性
縦書き変換ツールは存在するが、リアルタイムプレビュー・PNG出力・ルビ対応を1ページで完結させたものはほぼない。

## 20年後の視点
2045年に空間コンピューティングで縦書きテキストが3D空間に浮かぶインターフェースが普及する前段階として、縦書きレイアウトへの親しみを育てるツール。

## 備考
日本語縦書き文化の継承という文化的価値がある。フォントはGoogle Fonts日本語フォント（Noto Serif JP等）をCDNで読み込む。
