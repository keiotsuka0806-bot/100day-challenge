# モチベトラッカー 仕様書

## 概要
その日のやる気・集中度・気分を1〜5でチェックインし、折れ線グラフで推移を可視化するアプリ。「自分のコンディションのクセ」を発見できる。（トレンド反映: 開発継続のモチベ管理ニーズ）

## ターゲットユーザー
個人開発者・フリーランス・在宅ワーカーなど、自己管理が必要な人

## MVP機能一覧
- [ ] 毎日3項目を5段階評価（やる気・集中度・気分）
- [ ] 直近30日の折れ線グラフで推移表示
- [ ] 今日のチェックイン済みかどうか表示
- [ ] 一言メモを追加できる
- [ ] Firebase Auth + Firestore で永続化

## 将来拡張（MVP後）
- 睡眠時間・運動有無などのサブ指標
- 週次サマリーのプッシュ通知
- 「高モチベの日の傾向」分析

## 技術スタック
- フロントエンド: Vanilla JS
- データ: Firebase Firestore + Auth
- ホスティング: Firebase Hosting
- 特殊ライブラリ: Chart.js（CDN）

## データモデル
```
checkins/{uid}/days/{YYYY-MM-DD}
  - motivation: 1-5
  - focus: 1-5
  - mood: 1-5
  - memo: string
  - createdAt: timestamp
```

- ログイン画面
- メイン: 今日のチェックイン入力 + 30日グラフ
