# AI投資家のプレゼン道場（PitchDojo）

## 概要
30秒の売り込み（企画・自己PR・提案）を書く／話すと、性格の違うAI投資家3人が別々の角度から突っ込み、説得力を鍛える壁打ち道場。優しい人・数字の鬼・天邪鬼が同時に襲ってくる。

## MVP機能
- ピッチを入力→3人格が同時に別質問（「で、誰が困ってるの?」「数字は?」「もう有るよね?」）
- 弱点を突かれた箇所をハイライト、答え直すと説得力メーターが上がる
- "3人を黙らせた"最終ピッチをシェア

## 技術スタック
- フロントエンド: Vanilla JS / PWA / Vercel
- 特殊: Vercel関数+OpenAI(gpt-4o-mini)＋Web Speech API

## 難度
**中（半日）**

## 独自性
「決断カウンシル(decision-council)が意思決定を助けるのに対し、こちらは"自分が説得する側"を鍛える攻めの訓練」。プレゼンや面接前の人が、痛いところを先に潰せる。

## 20年後の視点
2045年、人は本番の前に必ずAIの厳しい観客で予行し、無防備なプレゼンは消えている。
