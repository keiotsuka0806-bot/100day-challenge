## 🎨 UI / UX トレンド

- **Calm UI（落ち着いたインターフェース）**：過剰なアニメ・エフェクトを廃し、「静かで明確」なデザインが主流。AI は「全知の操縦士」でなく「慎重な副操縦士」として設計する。
- **Glassmorphism 復活**：Apple の 2025 年デザインリニューアルを受け、磨りガラス風の半透明UIが再注目。
- **Bento Grid レイアウト**：サイズ・形の異なるブロックをお弁当箱のように並べたモジュラーデザイン。情報整理とビジュアルリズムを両立。
- **ダークモード × サイバーパンク美学**：深いブラック＋ネオンカラー（コーラル・ホログラフィックシルバー）の組み合わせ。
- **モーションは「オフ」にできること**：アニメは意図を持って使い、ユーザーが止められる設計が必須。
- **アクセシビリティが「機能」から「インフラ」へ**：ADHD・失読症・神経多様性ユーザーへの対応が基本要件に。

## ⚡ Firebase / Vercel アップデート

### Firebase
- **Firebase SQL Connect**（GA）：Cloud SQL for PostgreSQL と統合。リアルタイム同期・オフラインキャッシュ・SQL直書きが可能に。GraphQL 不要。
- **Firestore Enterprise**：ネイティブ全文検索、地理空間クエリ、JOIN相当のサブクエリ、DML が追加。
- **Phone Number Verification**（2026/5 GA）：SIM 経由でSMS不要の電話番号認証。10キャリア・6地域対応。コンバージョン率向上に期待。
- **Cloud Functions Dart サポート**（実験的）：Flutter 開発者が全スタックを Dart で統一可能に。
- **Firebase Studio**：Project IDX が統合され、AI フルスタックアプリ向けクラウド IDE として進化。
- **Firebase AI Logic**：Gemini 3.x 全モデル対応。Google Maps を使った Grounding で幻覚抑制。

### Vercel
- **Vercel Blob OIDC 認証**：短命な OIDC トークンが自動ローテーション。長期トークン（`BLOB_READ_WRITE_TOKEN`）が不要に。
- **Elastic Build Machines**：ビルド中のメモリ使用を監視し OOM 前に自動スケールアップ。
- **AI Gateway**：Alibaba の Qwen 3.7 Plus / Max が追加。
- **Vercel AI SDK**：Gemini Embedding・Deep Research プレビューモデルを追加。Azure OpenAI Responses API に Web 検索ツール統合。
