# TATSURO LP Workflow

## 天才集団

| 役割 | 見ること | 成果物 |
| --- | --- | --- |
| ブランド戦略家 | 誰に何を約束するLPか | ターゲット、ポジショニング、購入理由 |
| コピーライター | 一瞬で残る言葉か | H1、サブコピー、CTA、各セクション見出し |
| アートディレクター | 価格帯に見合う質感か | 写真、余白、色、タイポグラフィ |
| ウイスキー監修 | 味わい表現が自然か | 香り、味、余韻、飲み方の説明 |
| ECグロース担当 | 迷わず次の行動に進めるか | CTA、フォーム、導線、計測設計 |
| 法務・酒類販売監修 | 酒類広告として危険がないか | 年齢確認、注意書き、販売表現チェック |
| フロントエンド実装者 | 速く、崩れず、読めるか | HTML/CSS/JS、モバイル対応、PWA |

## ワークフロー

1. 目的を決める: 認知、予約、購入、問い合わせのどれを狙うか。
2. ブランド核を決める: 名前、価格帯、飲む場面、人格、禁止する表現。
3. ファーストビューを作る: 商品画像、H1、サブコピー、CTAを最初に固める。
4. 信頼情報を足す: 味わい、製法、容量、度数、飲み方、注意表示を整理する。
5. 導線を作る: フォーム、購入ボタン、SNS、問い合わせのいずれかを明確にする。
6. Genius Review 3ループ: 第一印象、専門性、共有/購入導線の順で直す。
7. 実機確認: iPhone幅、PC幅、画像読み込み、フォーム、年齢表示を確認する。
