# TATSURO LP Review

## 参照した共通点

- ファーストビューで商品名、ビジュアル、要点が同時に読める
- 見出しは短く、短いなら1行で終わらせる
- 強制改行は避け、必要な改行も記号でつながない
- 仕様は文章ではなく、度数・容量・飲み方として素早く出す
- ボトルと周辺情報を並べて見せる
- 酒類LPは年齢確認と責任ある飲酒表現を明示する
- 日本語LPは英日混在を減らし、ラベルの粒度を揃える

## 3回の会議

### 1回目

- 役割: ブランド戦略家 + コピーライター
- 論点: ヒーローが長く、主張が弱く見える
- 結論: 短い一文を1行で置く

### 2回目

- 役割: アートディレクター + ECグロース担当
- 論点: 余白より先に要素数が目につく
- 結論: 情報を減らし、帯でまとめる

### 3回目

- 役割: 法務・酒類販売監修 + モバイルUX
- 論点: 注意文とCTAの距離が遠い
- 結論: 年齢確認をフォーム直下に置く

### 4回目

- 役割: テイスティング担当 + プロダクト編集者
- 論点: 味わい情報のラベルが揺れている
- 結論: 日本語ラベルに揃える

### 5回目

- 役割: モバイルQA + 情報設計者
- 論点: 改行が人工的で、文章が不自然
- 結論: 強制改行を外し、短文は1行で見せる

## 今回の修正

- ヒーローを1行の短文に変更
- 直下に仕様帯を維持しつつラベルを日本語化
- テイスティングを日本語ラベルに再構成
- 製法と案内の見出しを短文化
- 強制改行を外して自然な行送りに寄せた
