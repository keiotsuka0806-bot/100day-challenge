# TATSURO LP Review

## 参照した共通点

- ファーストビューで商品名、ビジュアル、要点が同時に読める
- 文章は長くせず、香り・味・余韻のような短いブロックに分ける
- 仕様は文章ではなく、年数・度数・容量・飲み方として素早く出す
- ボトルとパック、または商品と関連情報を並べて見せる
- 酒類LPは年齢確認と責任ある飲酒表現を明示する

## 3回の会議

### 1回目

- 役割: ブランド戦略家 + コピーライター
- 論点: 文字量が多く、ブランドの格が下がって見える
- 結論: 約束を1行にして、説明を後ろへ送る

### 2回目

- 役割: アートディレクター + ECグロース担当
- 論点: 余白よりも先に「カードの塊」が目に入ってしまう
- 結論: チップ化しつつも、カード感を減らして編集的に見せる

### 3回目

- 役割: 法務・酒類販売監修 + モバイルUX
- 論点: 文章の折返しと注意文の置き場所
- 結論: モバイルで行長を削り、注意文はフォームの下に置く

### 4回目

- 役割: テイスティング担当 + プロダクト編集者
- 論点: 味わい情報が物語と分離している
- 結論: Color / Nose / Palate / Finish に再構成する

### 5回目

- 役割: モバイルQA + 情報設計者
- 論点: スペックとCTAが散っている
- 結論: ヒーロー直下にスペック帯を置き、CTAは1つを主役にする

## 今回の修正

- ヒーローを短い約束に変更
- 直下に仕様チップを追加
- テイスティングを4行に再構成
- 制作メモ的な章を公開ページから削除
- モバイルの行長をさらに縮めた
