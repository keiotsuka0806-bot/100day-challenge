# iPhoneで「買い物アプリを開くと自動で立ち上げる」設定

衝動ブレーキ本体はふつうのWebアプリなので、iOS標準の「ショートカット」に
一度だけオートメーションを設定すると、Amazonや楽天などを開いた瞬間に自動で立ち上がります。

## 手順
1. iPhoneの **ショートカット** アプリを開く
2. 下タブ **オートメーション** → 右上 **＋** → **個人用オートメーションを作成**
3. **App** を選ぶ
4. 「App」欄で見張りたい買い物アプリを選ぶ（複数可）
   - おすすめ: Amazon / 楽天市場 / Yahoo!ショッピング / メルカリ / ZOZOTOWN / Qoo10
   - 「**開いている**」にチェック → 次へ
5. **アクションを追加** → 「URLを開く」を検索して追加
6. URL欄に衝動ブレーキの公開URLを貼る（例: https://impulse-brake.vercel.app ）
7. 次へ → **「実行の前に尋ねる」をオフ**（すぐ立ち上げるため）→ 完了

これで対象アプリを開くたびに、先に衝動ブレーキが開き「何を買おうとしてる？」と尋ねます。

## 補足・限界
- **ブラウザ（Safari）で買う場合**は反応しません（アプリを開いた時だけ効く仕様のため）。主要ショップはアプリがあるので、アプリ経由に寄せると効果的です。
- **Android** の自動起動は専用アプリが必要で、現在は未対応（今後対応予定）。
- 強制はしません。あくまで「一拍おいて、貼るよう促す」ところまでです。
